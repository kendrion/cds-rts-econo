English:
--------
To compile the m4-files of the runtime system to header files, you need the GNU m4-Preprocessor. This is protected by the GPL and can be downloaded and used for free (http://www.gnu.org/software/m4).
Every starter packet of teh runtmie system and in every source tree the m4-Compiler is included. You can find all necessary stuff of m4-Compiler in the runtime tree under $\BuildUtils.

To compile a m4-file (Itf.m4 or Dep.m4) to a header file with a double click, the easiest way is to use the the Batch File $\BuildUtils\msys\bin\compile_m4.bat that must be used 
fpr every m4-file.

You can do this in the Windows-Wxplorer under the menu:

        Extras
        Folder options...
        Page: File types
        search for m4-file and press button "Change..." or with button "New" create a new entry for the ending "m4"
        Select $\BuildUtils\msys\bin\compile_m4.bat as the program to execute behind this file types 

Now you can compile every m4-file in the runtime tree by a simple double click! 






Deutsch:
--------
Damit man m4-Dateien des Laufzeitsystems zu Header Dateien compilieren kann, benötigt man den GNU m4-Präprozessor. Dieser unterliegt der GPL und ist frei verfügbar (http://www.gnu.org/software/m4).
Bei jedem Starterpaket und auch in jedem Sourcebaum des Laufzeitsystems ist der m4-Compiler mit dabei. Alles für den m4-Compiler findet man im LZS-Verzeichnis immer unter $\BuildUtils.

Um nun eine m4-Datei (Itf.m4 oder Dep.m4) zu einer Header Datei mit einem Doppelklick zu erzeugen, muss man für eine m4-Datei das Batch File $\BuildUtils\msys\bin\compile_m4.bat für die Ausführung hinterlegen.

Das macht man am einfachsten im Windows-Explorer unter dem Menü:

        Extras
        Ordneroptionen...
        Registerkarte: Dateitypen
        nach m4-Datei suchen und auf Button "Ändern..." gehen oder mit Button "Neu" einen neuen Eintrag für die Dateiendung "m4" erzeugen
        $\BuildUtils\msys\bin\compile_m4.bat als ausführbares Programm auswählen 

Nun kann man in jedem Verzeichnis des Laufzeitsystems mit einem Doppelklick eine m4-Datei zu einer Header Datei compilieren lassen!
