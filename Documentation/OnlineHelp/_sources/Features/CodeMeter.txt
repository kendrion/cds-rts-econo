
WIBU CodeMeter
==============

With this feature the support for WIBU CodeMeter is enabled. See `CodeMeter-Tutorial`_ for details.

.. _`CodeMeter-Tutorial`: ../_downloads/CODESYSControlV3_CodeMeter_Tutorial.pdf    

---------------------

The following runtime system component is necessary to support this feature:

- CmpCodeMeter: implements interface :ref:`group___cmp_code_meter_itf`
