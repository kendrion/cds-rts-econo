
WIBU CodeMeter - Legacy
=======================

With this feature the support for WIBU CodeMeter is enabled with the legacy version v1.7, which is the last version that is support
under Windows CE!
So for all other targets use the STandard WIBU CodeMeter feature!

See `CodeMeter-Tutorial`_ for details.

.. _`CodeMeter-Tutorial`: ../_downloads/CODESYSControlV3_CodeMeter_Tutorial.pdf    

---------------------

The following runtime system component is necessary to support this feature:

- CmpCodeMeter_legacy: implements interface :ref:`group___cmp_code_meter_itf`
