WIBU CodeMeter - Legacy
=======================

With this feature the support for WIBU CodeMeter is enabled with the legacy version v1.7, which is the last version that supports Windows CE!
So for all other targets use the Standard WIBU CodeMeter feature :ref:`feature_codemeter`!

See :ref:`tutorial_codemeter` for details.
