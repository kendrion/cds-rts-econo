

.. _opcua_server_security:

OPC UA Server security features
===============================

This feature enables the security features of the OPC UA server. All needed requirements are added by this features.

The following features are supported:

Encrypted communication
-----------------------
The encrypted communication profiles of the OPC UA sepcification are suppored. For security reasons the SHA1 based profiles
are not available by default. As a result the profile Basic256SHA256 is available. Both signed and signed + encrypted
communication is supported.

Security settings
-----------------
The OPC UA server registers a number of security settings to change the security configuration. See: :ref:`group___cmp_o_p_c_u_a_server_itf___security_settings`

Requirements
------------
- CmpSecurityManager: :ref:`group___cmp_security_manager_itf` (implemented by CmpSecurityManager)
- CmpCrypto: :ref:`group___cmp_crypto_itf` (implemented by CmpOpenSSL)
- CmpX509Cert: :ref:`group___cmp_x509_cert_itf` (implemented by CmpOpenSSL)
