Edge Gateway
============

The Edge Gateway is a component, that connects a gateway or runtime system with a CODESYS Automation Server.

The connection is done via the WebSocket protocol. Configuration is done by a CODESYS Edge Gateway PlugIn.

-----------------

- :ref:`feature__p_l_c_handler`

- :ref:`feature__encrypted_communication`
