
SVG Support
===========

The image format svg is not supported on all devices. 

- Windows Target Visualization: Supported

- Web Visualization: Supported    

- Linux Target Visualization: Must be provided by the QT installation

- WinCE Target Visualization: Not supported. 

| In cases where svg is not supported the svg images are automatically converted to png or bmp depending on the available image formats on the device. If png is available png is used.
