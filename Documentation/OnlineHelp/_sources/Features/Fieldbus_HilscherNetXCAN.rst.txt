Hilscher netX CAN
=================

With this feature you can support Hilscher cifX cards (https://www.hilscher.com/products/product-groups/pc-cards) or netX controller (https://www.hilscher.com/products/product-groups/network-controller)
with the CAN fieldbus protocol.

The basic functionality to support cifX or netX device is implemented in the :ref:`feature_hilscher_cifx` feature. So this feature is enabled implicitly.
