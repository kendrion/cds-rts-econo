Quick installation guide
------------------------
To quickly install and run your CODESYS Control runtime, copy it to your target through SCP or a similar protocol and execute it in an SSH session.

Copy the binaries to your target
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    | *Replace* ``root`` *with your target's username*
    | *Replace* ``192.168.0.1`` *with your target's IP address*
    | *This will create a new directory* ``codesys`` *in user root's home directory*

    .. code-block:: shell

        scp -r Platforms/Linux/Bin root@192.168.0.1:codesys

Running CODESYS Control runtime
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    | *The argument* ``-d`` *is highly recommended during development as it enables additional debug output*

    .. code-block:: shell

        ssh root@192.168.0.1
        cd codesys
        ./codesyscontrol -d CODESYSControl.cfg

Press Ctrl-C to cleanly shutdown CODESYS Control.
