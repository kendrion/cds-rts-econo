.. _glossary:

Glossary
========

**Trace Packet**: A trace or trace packet consists of a configuration and its
data sets. The configuration consists of a context (task), perhaps a
trigger, some options and at least one variable which should be traced
(record)

**Trace Record**: A trace record is a data set of one variable that should be
traced. This data set contains, the a ring buffer with a value/timestamp
pair and some options to record this variable
