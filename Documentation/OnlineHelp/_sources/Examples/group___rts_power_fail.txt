Detailed Description
--------------------



Example to handle power fail conditions.

USAGE:

- InitPowerFail() must be called out of CH_INIT3 hook

- ExitPowerFail() must be called out of CH_EXIT3 hook

- :ref:`CBHandlePowerFail() <group___rts_power_fail_1gad428db51f6f146dff91ec3a27f25370e>` is the callback function to handle power fail condition

RTS_RESULT CBHandlePowerFail (RTS_UI32 shutdownOption)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Callback function to handle power fail signal

**Parameters**

+---------+----------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | shutdownOption | Shutdown option. Depends on the hold time of the power supply SHUTDOWN_FAST: This should be used, if the power supply only holds less time. SHUTDOWN_STANDARD: This should be used, if the power supply holds enough time to stop application and save retains. SHUTDOWN_FULL: This should be used, if the power enough supply holds enough time to fully shutdown the runtime. |
+---------+----------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

**Returns:**

Error code
