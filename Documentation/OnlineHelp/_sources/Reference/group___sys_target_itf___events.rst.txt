.. _group___sys_target_itf___events_1gad5997ff57b6742e5b12b2dcc2f3bb933:

#define EVT_SysTarget_SetNodeName :ref:`MAKE_EVENTID <group___cmp_event_mgr_itf___convertmacro_1ga8f45ad6798a634d4e00140c12039007e>` ( :ref:`EVTCLASS_INFO <group___cmp_event_mgr_itf___eventclasses_1ga711863810010aec40693ca7c4643a025>` , 1)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Event is sent in function :ref:`SysTargetSetNodeName() <group___sys_target_itf_1ga25f49ebea17f5b5568187e4f3d2533e7>` before the new node name is stored in the settings. The following results can be returned by the event:

  - ERR_OK: Event is not handled. :ref:`SysTargetSetNodeName() <group___sys_target_itf_1ga25f49ebea17f5b5568187e4f3d2533e7>` will store the name in the settings.

  - ERR_NOTHING_TO_DO: The event handler has already stored the new node name somewhere, thus :ref:`SysTargetSetNodeName() <group___sys_target_itf_1ga25f49ebea17f5b5568187e4f3d2533e7>` shall not store the name. A later call to the OEM-specific implementation of :ref:`SysTargetGetNodeName() <group___sys_target_itf_1ga42c6ad369c3e427fc13c5287c360f013>` will return the new node name.

  - ERR_OPERATION_DENIED: The event handler denies to set the new node name. :ref:`SysTargetSetNodeName() <group___sys_target_itf_1ga25f49ebea17f5b5568187e4f3d2533e7>` shall not store the name.

  - ERR_NOT_SUPPORTED: Device does not support to store the node name. :ref:`SysTargetSetNodeName() <group___sys_target_itf_1ga25f49ebea17f5b5568187e4f3d2533e7>` shall not store the name.
