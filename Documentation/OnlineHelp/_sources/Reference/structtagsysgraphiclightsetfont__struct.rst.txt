Detailed Description
--------------------

Function to set the font for the following drawn texts.

**Parameters**

+---------+-----------------+------------------------------------------+
| [in]    | hDisplayContext | Handle to device context.                |
+---------+-----------------+------------------------------------------+
| [in]    | pszFontName     | Name of the font that should be created. |
+---------+-----------------+------------------------------------------+
| [in]    | iFontSize       | Size of the font.                        |
+---------+-----------------+------------------------------------------+
| [in]    | ulFontFlags     | The font flags.                          |
+---------+-----------------+------------------------------------------+
| [in]    | ulFontCharSet   | The font character set.                  |
+---------+-----------------+------------------------------------------+
| [in]    | ulFontColor     | The font color.                          |
+---------+-----------------+------------------------------------------+

**Returns:**

RESULT: Returns the runtime system error code (see CmpErrors.library).
