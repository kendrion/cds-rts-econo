Macro Definition Documentation
------------------------------
.. _group___sys_window_itf___window_creation_flags_1gac336baa5eba64185c4f26d2918fd6de4:

#define SYSWINCREATE_FLAGS_NONE 0x00000000
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

<type>Int</type>

For extensibility we define some flags that control some properties of a window. Any of these flags might be released to a special class of window (e.g. the edit control).

.. _group___sys_window_itf___window_creation_flags_1ga66cda37954778f7e656a4af68f8a48b4:

#define SYSWINCREATE_FLAGS_PARENTISOSHANDLE 0x00000001
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

<type>Int</type>

If this flag is set, then the parent window handle is an OS handle and not a window handle returned by the SysWindow-Component.

.. _group___sys_window_itf___window_creation_flags_1ga4d799070ef4c449a712733fcf7222046:

#define SYSWINCREATE_FLAGS_TEXT_ALIGN_LEFT 0x00000002
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

<type>Int</type>

If this flag is set, then the window or control is created with the text alignment LEFT.

.. _group___sys_window_itf___window_creation_flags_1ga55425e21e4fa2846908b1404f3aef49d:

#define SYSWINCREATE_FLAGS_TEXT_ALIGN_HCENTER 0x00000004
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

<type>Int</type>

If this flag is set, then the window or control is created with the text alignment CENTER.

.. _group___sys_window_itf___window_creation_flags_1gadcf5f22aacd04e680e229cfb0d086b12:

#define SYSWINCREATE_FLAGS_TEXT_ALIGN_RIGHT 0x00000008
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

<type>Int</type>

If this flag is set, then the window or control is created with the text alignment RIGHT.

.. _group___sys_window_itf___window_creation_flags_1gacf9d5a65548f407cc4b0ccc71d5ff4ae:

#define SYSWINCREATE_FLAGS_TOUCHHANDLING_ACTIVE 0x00000010
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

<type>Int</type>

If this flag is set, then the window will react to touch inputs if possible.

.. _group___sys_window_itf___window_creation_flags_1gacd36185d16c7ad7471180d89445e5cc4:

#define SYSWINCREATE_FLAGS_CREATE_HIDDEN 0x00000020
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

<type>Int</type>

If this flag is set, then the window will created hidden.

.. _group___sys_window_itf___window_creation_flags_1gad0a62348a46ac9553e7bb002b36c6691:

#define SYSWINCREATE_FLAGS_TRANSPARENT_BACKGROUND 0x00000080
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

<type>Int</type>

If this flag is set, then the window will use a transparent background. Please remark that this feature is only supported at the moment by Qt-Based implementations (an implementation supporting SYSWINSUPPORT_FLAG_TRANSPARENT_BACKGROUND)
