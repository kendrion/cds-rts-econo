Compiler switches to trigger special behaviour of this component.

- VISH_REMOTE_START_IN_COMM_CYCLE: Usually the remote targetvisualization starts it's communication in the component manager hook CH_INIT_TASKS. When this define is active, then this startup is delayed until the first call of the hook CH_COMM_CYCLE. Typically this can be used if other components (or startup code) want to attach to the event
