+---------+---------------+------------------------------------------------------------------------------------------+

| [in]    | pduData       | References the request data (without header)                                             |
+---------+---------------+------------------------------------------------------------------------------------------+
| [in]    | pduSendBuffer | Contains the length of the buffer for the reply data, and the buffer for the replay data |
+---------+---------------+------------------------------------------------------------------------------------------+

**Returns:**
