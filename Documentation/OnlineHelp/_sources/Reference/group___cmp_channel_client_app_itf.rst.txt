STATICITF_DEF int ClientAppHandleMessage (RTS_HANDLE hInstance, unsigned short wChannelHandle, PROTOCOL_DATA_UNIT pduData)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Called when a layer4 message has been received.

**Parameters**

+---------+----------------+---------------------------------------+
| [in]    | hInstance      | Handle to the instance                |
+---------+----------------+---------------------------------------+
| [in]    | wChannelHandle | The channel that received the message |
+---------+----------------+---------------------------------------+
| [in]    | pduData        | The message that has been received    |
+---------+----------------+---------------------------------------+

**Returns:**

error code

.. _group___cmp_channel_client_app_itf_1ga9306a585871739828b4705fa05044970:

STATICITF_DEF int ClientAppOnChannelError (RTS_HANDLE hInstance, unsigned short wChannelHandle, int nError)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Called by the NetClient component when a channel is closed by the server or a communication error occurs. This function is not called when the channel is closed by a call to NetClientCloseChannel.

**Parameters**

+---------+----------------+-------------------------------------+
| [in]    | hInstance      | Handle to the instance              |
+---------+----------------+-------------------------------------+
| [in]    | wChannelHandle | The closed channel                  |
+---------+----------------+-------------------------------------+
| [in]    | nError         | The cause why the channel is closed |
+---------+----------------+-------------------------------------+

**Returns:**

error code
