.. _group___cmp_file_transfer_itf___staticdefines_1ga1507f43f4b09cb2e0036af36833ad8c7:

#define FT_TEMP_FILE_EXTENSION ".tmp"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Extension for files during download sequence. This extension is removed at the end of the successful transfer or renamed into FT_BACKUP_FILE_EXTENSION, if origin file is occupied with a parallel running file transfer. NOTE: If this extension is defined empty "", no rename of the transferred file is done. FT_BACKUP_FILE_EXTENSION must be defined empty too!

.. _group___cmp_file_transfer_itf___staticdefines_1gac98cc21674c0a6abf05d8b4078fed25b:

#define FT_BACKUP_FILE_EXTENSION ".bak"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Extension for files after successful file download. The extension is removed, if the still running file transfer on the source file ends. NOTE: If this extension is defined empty "", no rename of the transferred file is done. FT_TEMP_FILE_EXTENSION must be defined empty too!
