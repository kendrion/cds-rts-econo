#define SYSETHERNETKEY_INT_LINUX_PROTOCOLFILTER "Linux.ProtocolFilter"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

filter for incoming packets by protocol type. Default is Ethercat Set Linux.ProtocolFilter=3 for no filtering at all(for profinet or ethnetIP). See linux/if_ether.h for valid filter values

.. _group___sys_ethernet_itf___settings_1gaee894e6e169e51994afc1f9104921fa7:

#define SYSETHERNETKEY_INT_LINUX_INTERFACE_FILTER "Linux.InterfaceFilter"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Interface names which are allowed for SysEthernet. If applied, no other names are accepted. Interface name list must be indexed, started with 0, e.g.: Linux.InterfaceFilter.0=eth0 Linux.InterfaceFilter.1=eth1
