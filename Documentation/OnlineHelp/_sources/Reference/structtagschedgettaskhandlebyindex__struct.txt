Detailed Description
--------------------

Function returns the task handle of a task specified by an index.

**Parameters**

+---------+---------+------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | pApp    | If an application is specified, only the task of this application is returned. If NULL, the task with the index in all tasks i returned. |
+---------+---------+------------------------------------------------------------------------------------------------------------------------------------------+
| [out]   | pResult | ERR_OK or Error code                                                                                                                     |
+---------+---------+------------------------------------------------------------------------------------------------------------------------------------------+

**Returns:**

Handle to the task
