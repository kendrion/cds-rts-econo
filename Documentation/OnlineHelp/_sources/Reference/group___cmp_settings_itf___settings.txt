.. _group___cmp_settings_itf___settings_1gaaf6f823709d6bd521fe24d84786eaaff:

#define CMPSETTINGS_KEY_STRING_FILE_REFERENCE "FileReference"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Setting to specify external configuration files: File reference list must be indexed, started with 0 or 1, e.g.: FileReference.0=[Configurationfile.cfg]

If the file does not exist, the file is created. You can specify sections, that should be created in this referenced file, by a comma separated list. e.g.: FileReference.0=[Configurationfile.cfg], CmpApp
