STATICITF_DEF RTS_HANDLE OpcUaProviderCreate (RTS_HANDLE hIOpcUaProvider, CLASSID ClassId, int iId, RTS_RESULT *pResult)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Create a new OPC UA provider instance. Is typically internally used by the provider itself

**Parameters**

+---------+-----------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | hIOpcUaProvider | Handle to the provider interface. Must be 0 and is filled automatically by calling the :ref:`CAL_OpcUaProviderCreate() <group___cmp_o_p_c_u_a_provider_itf_1ga2cf3c93e883ecfb41fb03abdb7a1e1cd>` macro! |
+---------+-----------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | ClassId         | ClassID of the driver. See "Class IDs" section in CmpItf.h or in the Dep.h file of the provider.                                                                                                        |
+---------+-----------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | iId             | Instance number of the provider                                                                                                                                                                         |
+---------+-----------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [out]   | pResult         | Pointer to error code                                                                                                                                                                                   |
+---------+-----------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

**Returns:**

Should return RTS_INVALID_HANDLE

.. _group___cmp_o_p_c_u_a_provider_itf_1ga56ea82957acf737dd40e83d61ad5ccda:

^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

**Parameters**

+---------+----------------+---------------------------------+
| [in]    | hOpcUaProvider | Handle to the provider instance |
+---------+----------------+---------------------------------+

**Returns:**

.. _group___cmp_o_p_c_u_a_provider_itf_1gae3472f26b23dce0fa1c322b256043fd9:

STATICITF_DEF OpcUa_StatusCode OpcUaProviderInitialize (RTS_HANDLE hProvider)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This function is called by the OPC UA server to initialize the provider. After the call to this function the provider should be ready to be used.
**Returns:**

Function returns OpcUa_Good if everything went fine. Return OpcUa_Bad if an error occurs.

.. _group___cmp_o_p_c_u_a_provider_itf_1gae92fc3defe26756884b5e6b3e13a8531:

STATICITF_DEF OpcUa_StatusCode OpcUaProviderCleanup (RTS_HANDLE hProvider)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This function is called by the OPC UA server to cleanup the provider. Resources should be clean up.
**Returns:**

Function returns OpcUa_Good if everything went fine. Return OpcUa_Bad if an error occurs.
