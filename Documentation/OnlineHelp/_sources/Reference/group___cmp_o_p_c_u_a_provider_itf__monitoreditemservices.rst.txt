.. _group___cmp_o_p_c_u_a_provider_itf__monitoreditemservices_1ga0b6fe6e876bcc86a97e22f4461e8fbea:

STATICITF_DEF OpcUa_StatusCode OpcUaProviderSampleMonitoredItem (RTS_HANDLE hProvider, OpcUaProvider_SampleMonitoredItemContext *pContext)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This function is used to delete a monitored item

This function is used to clean up the monitored items. The provider should clean up the provider specific informations created within the CreateMonitoredItem function and other stuff dedicated to this item.

The DataValues are freed by the server.

**Parameters**

+---------+----------+--------------------------------+
| [in]    | pContext | Information of item to sample. |
+---------+----------+--------------------------------+

**Returns:**

Function returns OpcUa_Good if everything went fine. Return OpcUa_Bad if an error occurs.
