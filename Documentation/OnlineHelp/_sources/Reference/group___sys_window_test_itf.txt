Detailed Description
--------------------

Interface of SysWindowTest.

SysWindowTest interfaces is intended to allow access to operating system methods that are necessary for automatic testing of window components. If you do not intend to do such tests on a platform, an implementation of this interface is not necessary!

Normally, components implementing this interface are bound very tightly to the implementation of the SysWindow-Interface.

RTS_RESULT SysWindowTestFindWindow (char *pszWindowClass, char *pszWindowTitle, RTS_HANDLE *phFoundWindow)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Function to search the operating system's window manager for a window by some criteria.

**Parameters**

+---------+----------------+--------------------------------------------------------------------------------------------------------------------------+
| [in]    | pszWindowClass | The name of the window class to search for                                                                               |
+---------+----------------+--------------------------------------------------------------------------------------------------------------------------+
| [in]    | pszWindowTitle | The name of the window title to search for. If the title of the window should be ignored, a value of NULL can be passed. |
+---------+----------------+--------------------------------------------------------------------------------------------------------------------------+
| [out]   | phFoundWindow  | The window handle (if one was found)                                                                                     |
+---------+----------------+--------------------------------------------------------------------------------------------------------------------------+

**Returns:**

The result code, ERR_OK if the search was successful and a window was found, another error code otherwise.

RTS_RESULT SysWindowTestTakeScreenShot (RTS_HANDLE hWindow, char *pszDestinationFileName, SysWindowTestScreenShotInfo *pScreenShotInfo)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Function to take a screenshot into a file from the passed window.

**Parameters**

+---------+------------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | hWindow                | The window to take a screen shot from.                                                                                                                                                          |
+---------+------------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | pszDestinationFileName | The file name of the screen shot that should be written.                                                                                                                                        |
+---------+------------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [out]   | pScreenShotInfo        | This pointer will be used to return information about the format of the screenshot. A pointer to a valid object must be passed; This object will then be filled with the necessary information. |
+---------+------------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

This method is optional (either by returnvalue ERR_NOTIMPLEMENTED or at compiletime) when the related SysGraphicimplementation implements the interface SysGraphicTestable

**Returns:**

The result code, ERR_OK if the screenshot was successful, another error code otherwise.

**Parameters**

+---------+---------------------+------------------------------------------------------------------------------------------------+
| [in]    | hWindow             | The window where the mouse event should be raised                                              |

| [in]    | xCoordinateInWindow | The x coordinate in window relative positions where the event should be raised.                |
+---------+---------------------+------------------------------------------------------------------------------------------------+

+---------+---------------------+------------------------------------------------------------------------------------------------+
