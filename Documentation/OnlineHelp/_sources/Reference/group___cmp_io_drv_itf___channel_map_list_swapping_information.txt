Macro Definition Documentation
------------------------------
.. _group___cmp_io_drv_itf___channel_map_list_swapping_information_1ga5c9f0f9de521797383ea8508a045651f:

#define CMLSI_BaseTypeMask 0x00FF
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Every Io driver needs information on the base data type of an io channel to perform a correct swapping operation.The basedata type and some additional swapping information can be found in the wDummy Byte of the ChannelMapList struct. This is the mask of the base type information

.. _group___cmp_io_drv_itf___channel_map_list_swapping_information_1gaa4bd981053a73f857acb8ac68a6eee44:

#define CMLSI_BaseDataTypeIsSet 0x0100
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

If this bit is set, the low byte of the wDummy word contains the base type information. The base type is of type TypClass3 of component SysInternalLib.

.. _group___cmp_io_drv_itf___channel_map_list_swapping_information_1gab8c1ec5c2f2d20cc04d5b2b7fea7f839:

#define CMLSI_SwappingEnabled 0x8000
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

If this bit is set, the driver has to determine, if the io channel must be swapped.
