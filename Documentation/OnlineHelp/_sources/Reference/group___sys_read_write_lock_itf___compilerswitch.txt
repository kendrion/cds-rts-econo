Detailed Description
--------------------



.. _group___sys_read_write_lock_itf___compilerswitch_1SysReadWriteLockItf_CompilerswitchSect0:

1
^

Compiler switch to enable/disable generic implementation. When SYSREADWRITELOCK_GENERIC is not defined, the implementation relies on the underlying operating system. If the switch is defined, the generic implementation is activated, indepentend of the underlying operating system.

- SYSREADWRITELOCK_GENERIC: This switch may be defined in sysdefines.h
