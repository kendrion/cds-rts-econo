Detailed Description
--------------------

Function to set the line style of the following objects.

**Parameters**

+---------+-----------------+----------------------------------------------------------+
| [in]    | hDisplayContext | Handle to device context.  Width OF the line as in Win32 |
+---------+-----------------+----------------------------------------------------------+
| [in]    | ulLineFlags     | Line flags. Defined in IEC-Code for Visualization        |
+---------+-----------------+----------------------------------------------------------+
| [in]    | ulLineColor     | Line color. Defined in IEC-Code for Visualization        |
+---------+-----------------+----------------------------------------------------------+

**Returns:**

*RESULT: Returns the runtime system error code (see CmpErrors.library).
