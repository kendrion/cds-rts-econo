.. _group___sys_flash_itf___settings_1gae88272e69b66e214213b518c9a7a8502:

#define SYSFLASH_KEY_INT_ERASE_BLOCK_SIZE "EraseBlockSize"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Max. size of a block, which is erased in one piece. If a block exceeds this size, the erase operation is splitted up into several calls of :ref:`SysFlashErase_() <group___sys_flash_itf_1ga21fd79f255440888df20a0a73f181d87>` , each limited to this value. Should be set to a size, which is erased in below 1 second, because the CommCycleHook is called after each block to keep the rest of the RTS alive.
