#define SRV_READ_USER_RIGHTS_MGMT 0x03
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Service group: SG_CMPUSERMGR

Service: SRV_READ_USER_RIGHTS_MGMT

Request tags:

  - Tag GroupDB Properties. (0..1) (Binary): Handle of the last object received.

Response tags:

  - Tag Object (0..n): An object of the user rights management.

    - Tag Serviceresult (0..1) (RTS_UI16): Errorcode of the operation

    - Tag GroupDB Properties. (0..1) (Binary): Handle of the last object transmitted. Send this within the next request to get the next bunch of objects.

    - Tag ObjectName (1) (RTS_UTF8STRING): The name of the object.

    - Tag ID (1) (RTS_UI32): ID of the object.

    - Tag ParentID (1) (RTS_UI32): ID of the parent object.

    - Tag UsedRights (0..1) (RTS_UI32): The evaluated rights of this object.

    - Tag ObjectProperty (0..1) (RTS_UI32): The properties of the object

    - Tag Group (0..n): A group that has configured rights on an object.

      - Tag GroupName (1) (RTS_UTF8STRING): The name of the group

      - Tag GarntedRights (0..1) (RTS_UI32): The granted rights on that object.

      - Tag DeniedRights (0..1) (RTS_UI32): The denied rights on that object.
