==============

.. contents::

Interface of CmpCryptMD5.

Detailed Description
--------------------

Interface for the MD5 hash feature.

ATTENTION: Don't use the MD5 hash in security relevant applications. MD5 is an unsafe algorithm because of existing collision tables!!
