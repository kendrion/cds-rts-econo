Detailed Description
--------------------



.. _group___cmp_io_mgr_itf___compilerswitch_1CmpIoMgrItf_CompilerswitchSect0:

1
^

Compiler switches to enable/disable single features in the component.

- CMPIECTASK_DONT_REGISTER_WATCHDOG_AT_SUPERVISOR: Switch to disable registering all hardware supported fieldbus watchdogs (e.g. Hilscher CIFX masters) at the CmpSupervisor! If this defined is set, the hardware supported fieldbus watchdogs are always enabled and cannot selectively be disabled via the CmpSupervisor!
