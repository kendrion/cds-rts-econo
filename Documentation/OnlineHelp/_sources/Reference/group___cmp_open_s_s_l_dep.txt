Detailed Description
--------------------



A component implementing the CmpTLS interface using the widly used open source OpenSSL library. OpenSSL is untouched, for easier patching new versions of OpenSSL in these component. CmpOpenSSL is able to encrype SysSocket based connections as well as the communicatin between CODESYS and the RTS using channels.
