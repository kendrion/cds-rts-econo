Detailed Description
--------------------

Parameters for the socket option SOCKET_SO_KEEPALIVE. NOTE: If one of the parameters is not supported, the result of :ref:`SysSockSetOption() <group___sys_socket_itf_1gaa342277e920cba3eda31a65cfd54c5b2>` is ERR_NOT_SUPPORTED. In this case, the corresponding result of the option contains the error result.
