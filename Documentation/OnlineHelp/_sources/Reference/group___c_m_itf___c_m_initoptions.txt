Specifies, if the Settingsfile is only used optional

.. _group___c_m_itf___c_m_initoptions_1gae839a8968bc34b256b5a523531a0c366:

#define RTS_CMINIT_OPTION_DONTCALLINITHOOKS UINT32_C(0x00000002)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

If this option is set, the CMInit sequence is stopped right after CH_INIT hook. To finish the startup sequence, you have to call :ref:`CMInitEnd() <group___c_m_itf_1ga83b61cae6ab87284f41f66db8d1d76d0>` afterwards. This can be used, if you have to do things first before before all other components, but you need here the runtime system components.
