#define CMPTARGETVISUSTUB_KEY_TARGETVISUPROCESS_IPCBASENAME "IpcBasename"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Setting that decides about the rate at which the target-visualization checks for updates of the content of the display. This value in ms is no real-time value, so there can be quite some jitter on this value. IMPORTANT: The according visualization handler stub must have configured the same base name !
