Detailed Description
--------------------

Execute all tasks, that are registered for the given event. The execution context may vary depending on the scheduler type. It might be executed synchronously in the context of the caller, or a system task (e.g. on-top of an OS) is just signaled to execute the task code.

Note, that the result ERR_NO_OBJECT might not be a real error. Depending on the system it might be OK, that no task was registered for this event.

**Parameters**

+---------+-----------+--------------------------------------------------------------+
| [in]    | hExtEvent | Handle to the external event, which tasks should be signaled |
+---------+-----------+--------------------------------------------------------------+

**Return values**

+---------------+---------------------------------------------+
| ERR_OK        | Event was signaled successfully             |
+---------------+---------------------------------------------+
| ERR_PARAMETER | The event handle was not found              |
+---------------+---------------------------------------------+
| ERR_NO_OBJECT | There was no task, registered on this event |
+---------------+---------------------------------------------+

**Returns:**

Error Code
