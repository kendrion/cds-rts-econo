====================
SysSocketEmbeddedDep
====================

.. contents::
    :local:
    :depth: 1



Detailed Description
--------------------

System specific implementation of the sockets interface for embedded devices. This component contains a minimal UDP stack just to satisfy the CoDeSys communication. It is no full replacement for an IP stack.
