.. _group___cmp_gw_client_itf___settings_1ga8388a53f8737be0e268916b28ef0a7b9:

#define GWCLIENTKEY_INT_INACTIVITY_TIMEOUT "InactivityTimeoutMs"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

<type>Int</type>

Setting to modify the default value of the Gateway inactivity timeout in ms. This default value is only used, if the Gateway connect is started by BeginConnectToGateway() instead of BeginConnectToGateway2(). If no piece of data (also no keep alive) is received or sent within this time, the GwClient closes the connection to the Gateway.
