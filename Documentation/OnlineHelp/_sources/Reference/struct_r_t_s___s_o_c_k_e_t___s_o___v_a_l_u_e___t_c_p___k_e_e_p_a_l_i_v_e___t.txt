Detailed Description
--------------------

Parameters for the socket option SOCKET_SO_KEEPALIVE. NOTE: If one of the parameters is not supported, the result of :ref:`SysSockSetOption() <group___cmp_blk_drv_tcp_dep___task_1gaa2c473b5715f4ed234ab62b8d0892c19>` is ERR_NOT_SUPPORTED. In this case, the corresponding result of the option contains the error result.
