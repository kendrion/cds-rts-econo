Detailed Description
--------------------

<SIL2>

Mapping information for a single channel. Every I/O-channel is described as a parameter, but with special meanings. The datatype of a channel can be simple (BOOL, BYTE, WORD, etc.) or array of simple types.

**Parameters**

+---------+----------------------+-----------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | pParameter           | Pointer to the parameter description                                                                                                          |
+---------+----------------------+-----------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | pbyIecAddress        | Pointer to the parameter in the process image on the IEC side                                                                                 |
+---------+----------------------+-----------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | wParameterBitOffset  | Bitoffset of the device                                                                                                                       |
+---------+----------------------+-----------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | wIecAddressBitOffset | Bitoffset of pbyIecAddress in the IEC process image                                                                                           |
+---------+----------------------+-----------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | wSize                | Size in bits of the channel                                                                                                                   |
+---------+----------------------+-----------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | wBaseTypeInformation | If this field is unequal zero, it is used for the base type information. With this information, a correct swapping operation can be executed. |
+---------+----------------------+-----------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | dwDriverSpecific     | Driver specific entry. Reserved for the usage in the driver.                                                                                  |
+---------+----------------------+-----------------------------------------------------------------------------------------------------------------------------------------------+
