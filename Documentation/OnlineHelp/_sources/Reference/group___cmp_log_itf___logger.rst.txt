^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Options of a logger (part of the configuration).

**Parameters**

+---------+--------------+----------------------------------------------------------------------------+
| [in]    | szName       | Name of logger                                                             |
+---------+--------------+----------------------------------------------------------------------------+
| [in]    | bEnable      | :ref:`Logger <struct_logger>` active/inactive                              |
+---------+--------------+----------------------------------------------------------------------------+
| [in]    | iType        | :ref:`Logger <struct_logger>` type. See above log types.                   |
+---------+--------------+----------------------------------------------------------------------------+
| [in]    | iFilter      | Filter for the logger entries to store. It is a combination of log classes |
+---------+--------------+----------------------------------------------------------------------------+
| [in]    | iMaxEntries  | Max number of possible log entries                                         |
+---------+--------------+----------------------------------------------------------------------------+
| [in]    | iMaxFileSize | Max file size, if logger content is stored in a file                       |
+---------+--------------+----------------------------------------------------------------------------+
| [in]    | iMaxFiles    | Max number of files, if logger content is stored in a file                 |
+---------+--------------+----------------------------------------------------------------------------+
| [in]    | pszPath      | Optional path for the logger file                                          |
+---------+--------------+----------------------------------------------------------------------------+
