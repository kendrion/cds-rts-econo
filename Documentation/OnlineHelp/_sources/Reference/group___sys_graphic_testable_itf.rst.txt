=====================
SysGraphicTestableItf
=====================

.. contents::
    :local:
    :depth: 1

Interface of SysGraphicTestable.

Detailed Description
--------------------

The interface SysGraphicTestable is intended for implementing automatic testability in a graphical component. Typically this interface can be implemented by components intended for embedded graphics (frame-buffer, light visualization). It is primarily meant as a simplification of SysGraphicNative to support automated testing.

RTS_HANDLE SysGraphicTestableCreate (RTS_RESULT *pResult)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Creates a handle to a test object that can be used in further calls. If you do not use this handle any more, release it using SysGraphicTestableDestroy

**Parameters**

+---------+---------+----------------------------------+
| [out]   | pResult | Optionally returns an error code |
+---------+---------+----------------------------------+

For Implementers: In fact nothing has to be created; it's merely a way for getting calls to initialize and shutdown.

**Returns:**

The handle to the test object or RTS_INVALID_HANDLE in case of errors.

.. _group___sys_graphic_testable_itf_1ga639ef0771afacaf2104dc9ae9da6ff83:

RTS_RESULT SysGraphicTestableDestroy (RTS_HANDLE hTestable)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Releases a handle to a test object that was returned by SysGraphicTestableCreate.

**Parameters**

+---------+-----------------+---------------------------------------+
| [in]    | hDisplayContext | The native display-context to release |
+---------+-----------------+---------------------------------------+

**Returns:**

An error code
