#define CMPSECCHANNEL_KEY_STRING_CERTHASH "CertificateHash"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

<type>String</type>

Hash of the last used certificate. This is used to restore an expired certificate after the systems reboot. This setting is managed by the CmpSecureChannel and must not be set by hand.
