Detailed Description
--------------------

Checks user access rights on the specified object (authorization)! :return: Error code + ERR_OK: If user has the requested rights on the object or if no user management is configured + ERR_PARAMETER: If user is not available or the object does not exist + ERR_FAILED: If something failed retrieving the access rights + ERR_NO_ACCESS_RIGHTS: Explicitly denied rights + ERR_INVALID_HANDLE: hUser is invalid or unknown
