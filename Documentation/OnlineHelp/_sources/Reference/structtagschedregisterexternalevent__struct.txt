Detailed Description
--------------------

Register an external event, which might be used later by event tasks. You should register those events during the system startup, so that the scheduler knows all events when the application is loaded.

Valid Hook: CH_INIT2

Note: the parameter pszExtEventName needs to be constantly allocated. It might either be a constant string, a global string variable, or a dynamically allocated string. But the memory needs to be valid during the life time of the event handle.

**Parameters**

+---------+-----------------+-------------------------------------------------------------------------------------------------------------------+
| [in]    | pszExtEventName | Name of the event, as it was specified in the device description. This parameter needs to be constantly allocated |
+---------+-----------------+-------------------------------------------------------------------------------------------------------------------+
| [inout] | pResult         | Error Code                                                                                                        |
+---------+-----------------+-------------------------------------------------------------------------------------------------------------------+

**Return values**

+--------------+--------------------------------------------------------------+
| ERR_OK       | Event was registered successfully                            |
+--------------+--------------------------------------------------------------+
| ERR_NOMEMORY | There was not enough memory to register the new event handle |
+--------------+--------------------------------------------------------------+

**Returns:**

handle to the event
