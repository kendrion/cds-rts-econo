Macro Definition Documentation

.. _group___sys_graphic_itf___antialiasingmodes_1ga03b815f3c36c31aefaa7b2c145b01ff1:

#define SYSGRAPHIC_ANTIALIASING_NONE 0x00000000
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This mode for the antialiasing feature means that antialiasing is completely disabled. This is the default for newly created device contexts.

.. _group___sys_graphic_itf___antialiasingmodes_1ga5ed5df83408a4475bb178161ed9e1852:

#define SYSGRAPHIC_ANTIALIASING_LOWQUALITY 0x00000001
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This mode for the antialiasing feature means that antialiasing in a lower quality (better performance) is enabled

.. _group___sys_graphic_itf___antialiasingmodes_1ga6c8dc2fd44c3c05f3abcfd07671f7d37:

#define SYSGRAPHIC_ANTIALIASING_HIGHQUALITY 0x00000002
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This mode for the antialiasing feature means that antialiasing in a high quality (worse performance) is enabled
