.. _group___cmp_web_server_handler_v3_itf___settings_1gabe6071bd46970854b7911fcab41099f3:

#define CMPWEBSERVERHANDLERV3_KEY_WEBSERVER_ALLOW_FILE_TRANSFER_SERVICES "AllowFileTransferServices"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Allow file transfer services in the webserver handler. Attention if the webserver could be accessed by hostile traffic: If this setting is activated (!= 0) a runtime user management should be established to protect access to files. Additionally the usage of the setting "ForceFilePath" of the component "SysFile" is recommended to restrict access to folders outside the runtime system environment.
