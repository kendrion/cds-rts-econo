#define NETXCMD_KEY_NETID "NetId"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

<type>Int</type>

Boad number of the channel with the specified NetID. This setting is used, only if there is no setting for the board and channel number in the io configuration. Example: NetId.0.Board=0 // Channel with NetId = 0 is on Board 0 If board and channel number is not set, board 0 channel 0 is used for NetId 0.
