| [in]    | useTempSettings | The connection relevant settings like address or name will be taken from settings prefixed with the prefix "TEMP." instead of the original name if this value is TRUE. +---------+-----------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

This call will fail if the remote target-visualization is already running.

error code
