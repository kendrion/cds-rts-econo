.. _group___cmp_app_itf___resetoptions_1ga28999c8825551c663d7af383ffabae84:

#define RTS_RESET 0
^^^^^^^^^^^^^^^^^^^

Warm reset. All gobal data except retain data is reset to their default values.

.. _group___cmp_app_itf___resetoptions_1gafc41a6ad9f2c1de8528201e8448cde7f:

#define RTS_RESET_COLD 1
^^^^^^^^^^^^^^^^^^^^^^^^

Cold reset. All gobal data AND retain data is reset to their default values.

.. _group___cmp_app_itf___resetoptions_1gabbd2c3a826e88c0e2d23e57a5a9860a4:

#define RTS_RESET_ORIGIN 2
^^^^^^^^^^^^^^^^^^^^^^^^^^

Origin reset. Delete the application, delete all application files (bootproject, etc.), reset all gobal and retain data. After this command, the controller doesn't know anything about the application.

.. _group___cmp_app_itf___resetoptions_1ga71a5184e7090ca4ec8d6609ef4c09d57:

#define RTS_RESET_ORIGIN_DEVICE 3
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Origin reset device. Delete all applications, delete all application files (bootproject, etc.), reset all gobal and retain data. Clear $visu$ and $PlcLogic$ folders. After this command, the controller doesn't know anything about any application.
