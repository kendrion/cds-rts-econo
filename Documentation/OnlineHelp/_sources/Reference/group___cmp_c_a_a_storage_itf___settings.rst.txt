.. _group___cmp_c_a_a_storage_itf___settings_1gad8e8009a9222ee16d230117bb2cc055f:

#define CMPCAASTORAGE_KEY_MEMORY_LIMIT_HINT "MemoryLimitHintInMB"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Setting definining a hint for the maximal memory consumption of CmpCAAStorage. A value of 0 will disable this limitation.

The database system will attempt to keep the memory utilization below this limit. Nevertheless instead of raising "Out of Memory" errors, the limit might be exceeded. Apart from the memory affected by this value there might be some relatively small additional allocations (eg. for open databases, tables, cursors) that are not taken into account by this setting.

.. _group___cmp_c_a_a_storage_itf___settings_1ga23970a12dfffadfa4962ec5972c13e3a:

#define CMPCAASTORAGE_VALUE_MEMORY_LIMIT_HINT_DEFAULT 64
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. _group___cmp_c_a_a_storage_itf___settings_1ga6631ae132aaa75a8b17e588a8a2fc00b:

#define CMPCAASTORAGE_KEY_MEMORY_LIMIT_HARD_PERCENT "MemoryLimitHardPercent"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Setting defining a hard limit of the memory consumption of CmpCAAStorage in percent on top of MemoryLimitHintInMB. The value will be interpreted relatively to MemoryLimitHintInMB. If you specify a value of 20 then the hard limit will be 1.2 times the value of MemoryLimitHintInMB. A value below zero will disable this hard limitation.

If the database system should require more memory than allowed by this setting, "Out of Memory" errors will be raised.

.. _group___cmp_c_a_a_storage_itf___settings_1gaae66e99c1ba7e0df9e6831f4c5e5233a:

#define CMPCAASTORAGE_VALUE_MEMORY_LIMIT_HARD_PERCENT_DEFAULT 20
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

#define CMPCAASTORAGE_KEY_PAGE_CACHE_SIZE "PageCacheSize"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Setting defining the suggested maximal number of database pages that are kept in memory at a time per database conection.

Changing this setting might affect the performance of the database as it might increase / decrease the number of accesses to the filesystem.

.. _group___cmp_c_a_a_storage_itf___settings_1ga59bda905fbe3842792feffabfff46e24:

#define CMPCAASTORAGE_KEY_MEMORY_DIAGNOSIS_INTERVAL "MemoryDiagnosisInterval"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Setting that activates log messages printing the current memory consumption of the underyling database system of CmpCAAStorage. A value of 0 will disable this logging.

The value is interpreted as interval in seconds between logging memory statistics. The log messages are displayed with logclass debug.
