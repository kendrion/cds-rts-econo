#define CDKEY_INT_ADD_TIMESTAMP_FILENAME "AddTimestampToFilename"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Setting to enable/disable the addition of a timestamp to the filename. Activating this option also means that several Core Dump files might be stored on the PLC. This increases storage size requirements. 1: Add timestamp 0: Don't add timestamp (default)
