#define RTS_SUPERVISOR_FLAG_WATCHDOG 0x00010000
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This is a watchdog (consumer), that uses the supervisor to protect the controller! NOTE: This consumer can be disabled via the standard function :ref:`SupervisorOperationDisable() <group___cmp_supervisor_itf_1gacd0be713c20c1ee14e63fd69c082fbfc>` and so this watchdog will not expire, if any of the supervisor operations will fail!
