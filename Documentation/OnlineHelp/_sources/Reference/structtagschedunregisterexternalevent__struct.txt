Detailed Description
--------------------

Unregister an external event, which was registered by :ref:`SchedRegisterExternalEvent() <_cmp_schedule_itf_8h_1a880b7a6a07983c220103310b8c42199c>` before.

This function should be called on system shutdown

Valid Hook: CH_EXIT2

**Parameters**

+---------+-----------+------------------------------------------------------------+
| [in]    | hExtEvent | Handle to the external event, that should be unregistered. |
+---------+-----------+------------------------------------------------------------+

**Return values**

+---------------+--------------------------------------------+
| ERR_OK        | Event was deregistered successfully        |
+---------------+--------------------------------------------+
| ERR_PARAMETER | The event handle was not registered before |
+---------------+--------------------------------------------+

**Returns:**

Error code
