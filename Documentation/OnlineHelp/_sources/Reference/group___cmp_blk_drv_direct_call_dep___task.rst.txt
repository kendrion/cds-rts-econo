#define RTS_TASKNAME_BLKDRVDIRECTCALL "BlkDrvDirectCall"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Block driver DirectCall communication task. The priority of this task should be above the priority of all other block driver tasks and above the priority of the task which calls the CH_COMM_CYCLE. This ensures, that the intermediate buffer is always cleared, before the next block arrives.
