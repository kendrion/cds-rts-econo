Detailed Description
--------------------

WARNING! If you pass this structure to the IecVarAccess or CmpIecVarAccess, it MUST be initialized with IIecVarAccess9.VarAccInitVarInfo() or :ref:`IecVarAccInitVarInfo() <group___cmp_iec_var_access_itf_1ga722faf6483e1c0cd61756b40902b2391>` before usage, and it MUST be cleaned up with IIecVarAccess9.VarAccExitVarInfo() or :ref:`IecVarAccExitVarInfo() <group___cmp_iec_var_access_itf_1ga97b0fe9ad9fd0abb7adb9011b834165f>` . Weird side effects including crashes and memory leaks will result if you do not adhere to this rule.
