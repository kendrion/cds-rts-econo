^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Everyone group. Every user is automatically member of this group! NOTE: By default, this group has read-only access to the objects. But this can be deactivated, so every user must be a member of one of the groups above and have no access right by default!
