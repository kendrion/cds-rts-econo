Detailed Description
--------------------

Enables the trigger of a trace packet.

.. note:: This function should be called if the trigger is currently disabled. If the trigger has already fired, call |TraceMgrPacketResetTrigger| to reset the trigger, instead.

:return: Returns the runtime system error code (see CmpErrors.library):

- ERR_PARAMETER if hPacket is not a valid packet handle
