#define CMPRETAIN_DELETEOPTION_REMOVE 0
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Only remove retain memory from retain management

.. _group___cmp_retain_itf___deleteoption_1gaeda2f74460c5d6f6a8fb4e4860b7968d:

#define CMPRETAIN_DELETEOPTION_ALL 1
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Release retain memory, mark memory as free and call :ref:`RetainUpdate() <group___cmp_retain_itf_1ga5e4750cb069c93892ada509d08dea350>` to mark reserved segment as free! Complete retain memory is deleted!

.. _group___cmp_retain_itf___deleteoption_1gaf4a87d68ad10c1cb8415ec1040e9a673:

#define CMPRETAIN_DELETEOPTION_REMOVE_AND_FREE 2
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

remove retain memory from retain management and mark as free. Don't delete retain file!

.. _group___cmp_retain_itf___deleteoption_1gabd2de0326f8a77826a87197f1003d6a7:

#define CMPRETAIN_DELETEOPTION_REMOVE_AND_RESERVE 3
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

remove retain memory from retain management and mark used segments as reserved. Don't delete retain file!
