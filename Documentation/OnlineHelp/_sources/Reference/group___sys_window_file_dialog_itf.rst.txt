======================
SysWindowFileDialogItf
======================

.. contents::
    :local:
    :depth: 1

Interface of SysWindowFileDialog.

Detailed Description
--------------------

The SysWindowFileDialog interface is designed to have access to a file dialog that is displayed when a screen is available.
