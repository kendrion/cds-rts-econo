.. _group___cmp_visu_handler_remote_itf_1gacc2ca10c16d2b1f238db2d3a282dcf79:

typedef RTS_BOOL( * PFVishRemoteProvideCredentials)(RTS_HANDLE hCallerData, char *pszUserBuffer, RTS_SIZE userBufferSize, char *pszPasswordBuffer, RTS_SIZE passwordBufferSize)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This function will be called if it is necessary to login to the plc or a previous login attempt failed.

**Parameters**

+---------+--------------------+----------------------------------------------------------------------+
| [in]    | hCallerData        | An arbitrary object that will be forwarded to the callback functions |
+---------+--------------------+----------------------------------------------------------------------+
| [in]    | pszUserBuffer      | A buffer for the username. Will not be NULL.                         |
+---------+--------------------+----------------------------------------------------------------------+
| [in]    | userBufferSize     | The size of pszUserBuffer in bytes.                                  |
+---------+--------------------+----------------------------------------------------------------------+
| [in]    | pszPasswordBuffer  | A buffer for the password. Will not be NULL.                         |
+---------+--------------------+----------------------------------------------------------------------+
| [in]    | passwordBufferSize | The size of pszPasswordBuffer in bytes.                              |
+---------+--------------------+----------------------------------------------------------------------+

**Returns:**

Return FALSE to use the credentials. Return TRUE to cancel the surrounding operation.

.. _group___cmp_visu_handler_remote_itf_1ga717571fa8ad0062d28fa1ed4683d1952:

RTS_RESULT VishRemoteConnect (RTS_BOOL useTempSettings)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This function can be called to start the remote targetvisu in a delayed way. Typically this is done when the event EVT_CmpVisuHandlerRemote_CheckAutoStartup is handled to delay the initial startup.

**Parameters**

+---------+-----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | useTempSettings | The connection relevant settings like address or name will be taken from settings prefixed with the prefix "TEMP." instead of the original name if this value is TRUE. The idea is that temporary settings can be realized without overwriting possibly existing persistent settings. This call will fail if the remote targetvisu is already running. |
+---------+-----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

**Returns:**

error code

.. _group___cmp_visu_handler_remote_itf_1ga9ebf3f03d1644fd0318a107b190ac782:

RTS_RESULT VishRemoteReadApplications (RTS_HANDLE hCallerData, PFVishRemoteProvideCredentials pfProvideCredentials, PFVishRemoteNotifyApplicationName pfAppCallback, RTS_BOOL bReadTempSettings)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This function reads the available applications from the connected PLC and returns them using the configured callback functions.

**Parameters**

+---------+----------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | hCallerData          | An arbitrary object that will be forwarded to the callback functions                                                                                                                |
+---------+----------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | pfProvideCredentials | The callback function that will be called in case a login is required. Must not be NULL.                                                                                            |
+---------+----------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | pfAppCallback        | The callback function that will be called for each application on the connected PLC. Must not be NULL.                                                                              |
+---------+----------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [in]    | bReadTempSettings    | If TRUE then temporaray settings (beginning with "Temp.") are read from the configuration file are read, otherwise the "normal" settings The callbacks will be called synchronously |
+---------+----------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

**Returns:**

error code
