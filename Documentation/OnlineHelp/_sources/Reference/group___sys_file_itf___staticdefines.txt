#define SYSFILE_TEMP_BUFFER_SIZE 256
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Size for temporary buffer e.g. to calculate CRC. If size is larger than 256, buffer is allocated from heap!

.. _group___sys_file_itf___staticdefines_1ga96fc7165b997ecf231292fcd45dc4318:

#define SYSFILE_INVISIBLE_FILENAME_PREFIX "."
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

File name prefix to make a file invisible e.g. for the filetransfer dialog. This prevents the corresponding file to be overwritten/read/compromized from outside.

.. _group___sys_file_itf___staticdefines_1ga847b50a3be7afa100bcd75824bbc7e26:

#define SYSFILE_MAP_SECTION_NAME "SysFileMap"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This is the name for a section which contains the name, CRC and size of a file. Is used as a cache for these values. Improves for example the performance of the file transfer. SYSFILE_DISABLE_FILE_CACHE disables this feature. NOTE: It is recommended to use a filereference in the configuration, to move these entries into a separate cfg-file. e.g.: [CmpSettings] FileReference.0=SysFileMap.cfg, SysFileMap

.. _group___sys_file_itf___staticdefines_1ga9dc1103eb6c7feefe3d32910e8a03e32:

#define SYSFILE_KEY_INT_PLCLOGIC_PREFIX "PlcLogicPrefix"
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Setting to configure the plc logic placeholder prefix that is used now by all application related files. -1 - old behaviour: flat file structure (default) 0 - old behaviour: flat file structure, but Reset Origin Device will change that to 1 1 - new folder structure: $PlcLogic$ + <application> + $trend$ + $alarm$ + $ac_persistence$ + $visu$
