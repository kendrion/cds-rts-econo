#define BCT_START 0x00000001
^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Type of the bus cycle to differentiate between:

  - BCT_START: Bus Cycle is executed at the beginning of the task cycle

  - BCT_END: Bus Cycle is executed at the end of the task cycle
