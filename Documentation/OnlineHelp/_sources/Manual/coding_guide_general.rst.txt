General
-------

1. Implementation must be done in standard ANSI-C. We require at least ANSI C99-support of the C-Compiler (see :ref:`coding_guidelines_c99_support` for details)!
2. Switch on the highest warning level in the compiler or use strict ANSI-C
3. Every component must implement the standard component interface to be integrable in the runtime system (see :ref:`arch_comp_itf_architecture`).
4. Coding Style:
