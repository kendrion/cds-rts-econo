.. _arch_error_codes:

Error Codes
-----------

All functions in the runtime system return an error code. This error
code has the type RTS\_RESULT. All error codes can be found in
CmpErrors.h.

Standard error codes that can be used in each component are defined at
the beginning of CmpErrors.h. Each error code has a prefix with ERR\_.

Each kernel component can specify its own error codes in CmpErrors.h.

Each OEM components can specify its own error codes in the range f
0x8000..0xFFFF.
