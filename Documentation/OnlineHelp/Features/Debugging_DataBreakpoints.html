Debugging - Embedded

Debugging - DataBreakpoints¶

This feature enables the support for data breakpoints. Attention: this is only available on the CODESYS Control Win runtime system.

Required features¶

  • Debugging - Full

Required target settings¶

The following target settings are necessary for this feature:

  • data_breakpoints_supported

  • max_data_breakpoint_number

  • max_data_breakpoint_size

Required components¶

The following runtime system components are necessary for this feature:

  • SysCpuBreakpoints
