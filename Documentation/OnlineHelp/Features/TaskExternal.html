7.2.1.15. External event tasks¶

With this feature you can specify external event triggered IEC tasks. This feature is to activate the event for the IEC taskconfiguration.

The following targetsetting specifies the list of external event names. Is must be filled as needed in the device description:

externalevents

Additionally you have to specify the maximum number of possible external exent triggered IEC tasks:

maxexternalevents

Required features¶

  • Base device description feature

Required target settings¶

The following target settings are necessary for this feature:

  • supportexternal
