Remote Target Visualization¶

The following target settings are necessary for this feature:

  • remotetargetvisualization

The following runtime system components are necessary in the controller to which the remote target visualization should be connected.

  • CmpBitmapPool

  • CmpDynamicText

  • CmpVisuHandler

  • CmpVisuServer

The following runtime system interfaces are implemented by the components:

  • CmpBitmapPoolItf

  • CmpDynamicTextItf

  • CmpVisuHandlerItf

  • CmpVisuServerItf
