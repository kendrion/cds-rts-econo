Using the QT5 library for the linux target visualization¶

When this feature is activated, Qt uses the new WebEngine control.

Required features¶

  • Visualization

Required target settings¶

The following target settings are necessary for this feature:

  • visualization_Linux_QTWebControl

  • visualization_QNX_QTWebControl
