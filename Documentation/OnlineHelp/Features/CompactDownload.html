7.2.1.7. CompactDownload¶

We have to use this feature to support download IEC-code into CmpAppEmbedded.

Required features¶

  • Base device description feature

Required target settings¶

The following target settings are necessary for this feature:

  • compact_download

  • constants-in-own-segment

  • code-segment-header-size

  • memory-allocation-callback

  • Device_gvl_editor_extended_persistent_variables

Required components¶

The following runtime system components are necessary for this feature:

  • CmpAppEmbedded
