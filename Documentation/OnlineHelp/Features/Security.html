7.2.2.10. Security Features¶

More and more controllers and plants are networked with each other and so there is an increasing need to improve all controllers regarding security aspects!

Here we describe all currently available security features of the runtime system.

Overview¶

This feature bundle enables all currently available security features at once.

Two components are the most central security elements:

  • CmpOpenSSL

  • CmpSecurityManager

The feature Security Manager is implicitly added with this feature!
