SH Codegenerator¶

SysCpuHandling implementation notes¶

  • SysCpuGetInstancePointer():

Function to get the instance pointer on the basis of the base pointer. Both the instance pointer and the base pointer are only valid and used in the CODESYS application context. The offset is specific to the platform and is 8 byte for the SH codegenerator.

An example implementation may be found in Platforms/SysTemplates/Sys/SysCpuHandlingOS.c of the runtime system toolkit.

Required features¶

  • feature__c_p_u

Required target settings¶

The following target settings are necessary for this feature:

  • set_next_statement_supported

  • codegenerator_CPU

  • codegenerator-guid

  • codegenerator_backend-guid

  • Motorola Byte Order

  • pack-mode
