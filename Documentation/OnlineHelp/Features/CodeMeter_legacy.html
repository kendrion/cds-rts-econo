7.2.2.3. WIBU CodeMeter - Legacy¶

With this feature the support for WIBU CodeMeter is enabled with the legacy version v1.7, which is the last version that supports Windows CE! So for all other targets use the Standard WIBU CodeMeter feature WIBU CodeMeter!

See CodeMeter Tutorial for details.

Required features¶

  • Base device description feature

Required components¶

The following runtime system components are necessary for this feature:

  • CmpCodeMeter_legacy
