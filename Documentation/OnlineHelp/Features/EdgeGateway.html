Edge Gateway¶

The Edge Gateway is a component, that connects a gateway or runtime system with a CODESYS Automation Server.

The connection is done via the WebSocket protocol. Configuration is done by a CODESYS Edge Gateway PlugIn.

Required features¶

  • PLCHandler

  • Encrypted communication

Required components¶

The following runtime system components are necessary for this feature:

  • CmpEdgeGateway

  • CmpWebSocketClient

  • CmpAsyncMgr

  • SysSemCount

Compiler Defines¶

  • Platforms/All/COMPILER_DEFINES: NO_PARAMETERLIST_IN_MACROS

  • Platforms/All/COMPILER_DEFINES: ARTI_LINKINMODULE

  • Platforms/All/COMPILER_DEFINES: ARTI_SWAP_PROTOCOL_HEADER

  • Platforms/All/COMPILER_DEFINES: GWCLIENT3_LINKINMODULE

  • Platforms/Linux/CFLAGS_FEATURES: -DPLC_HANDLER_COMPONENT

  • Platforms/Linux/CXXFLAGS_FEATURES: -DPLC_HANDLER_COMPONENT

  • Platforms/Linux/CFLAGS_FEATURES: -DCMPPLCHANDLER_DISABLE_EXTREF

  • Platforms/Linux/CXXFLAGS_FEATURES: -DCMPPLCHANDLER_DISABLE_EXTREF

  • Platforms/Windows/COMPILER_DEFINES: CMPLPLCHANDLERMAIN_NOTIMPLEMENTED
