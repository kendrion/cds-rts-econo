1.7.2.1. AttributesÂ¶

There are some attributes for the device that can be assigned to the <Device> tag.

Attribute Description
showParamsInDevDescOrder If attribute is false or missing, then the parameter are shown in the generic device configuration editor in the order of the parameter id. If attribute is true, then the parameters are shown in the order of the position inside the device description.
hideInCatalogue If attribute is true, then the device will not be shown in the add device dialogue of CODESYS. It could be added by a plugin or device scan.

Example:

<Device showParamsInDevDescOrder="true">

Note

See attributes of element “DeviceDescription/Device” in device_description_schema for details.
