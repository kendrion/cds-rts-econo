1.7.3.5. Save and Restore Changed IO Configuration ParametersÂ¶

With the following setting for the CmpIoMgr component in the configuration file of the runtime, you can activate to store all changed parameter values in the file “IoConfig.”ar”:

[CmpIoMgr]
StoreChangedParameters=1

This file is written always if a parameter has been changed online or via IoMgrWriteParameter().

All changed parameters are restored at the next bootup, right after the bootproject with the IO-Configuration is loaded.
