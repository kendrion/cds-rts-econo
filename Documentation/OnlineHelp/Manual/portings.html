1.4. PortingsÂ¶

The runtime system is written completely portable. With the system interfaces, it can be ported very easy to an operating system or a special processor. The following portings are still available. Operating systems or processors/controllers that are not on the list can be provided on request.

We provide the following standard single license products, which can be optained from the CODESYS Store:

  • CODESYS Control for BeagleBone SL
  • CODESYS Control for emPC-A/iMX6 SL
  • CODESYS Control for IOT2000 SL
  • CODESYS Control for Linux SL
  • CODESYS Control for PFC100 SL
  • CODESYS Control for PFC200 SL
  • CODESYS Control for Raspberry Pi SL
  • CODESYS Control Linux MC SL
  • CODESYS Control RTE SL
  • CODESYS Control RTE SL (for Beckhoff CX)
  • CODESYS Control Win MC SL
  • CODESYS Control Win SL
  • CODESYS HMI SL

Beneath the standard single license products we provide runtime toolkits for OEM customer to built their own runtime system.

For the following platforms we provide perpared standard toolkits. If your platform is not in this list, please contact us:
