Name: runtime_identification\version

Default Value: 1.0.0.0

Recommended Value: 1.0.0.0

XML Type: string

dotNet Type: System.String

Example:

<ts:section name="runtime_identification">
      ...
    <ts:setting name="version" type="string" access="visible">
        <ts:value>1.0.0.0</ts:value>
    </ts:setting>
      ...
</ts:section>

Documentation:

This is the official 3S runtime version. Can be used by CODESYS
PlugIns to check available functionality of the runtime system! So
this version must be adapted always, if a new firmware version is
integrated on the target!
