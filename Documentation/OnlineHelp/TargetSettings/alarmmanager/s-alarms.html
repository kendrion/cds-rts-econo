Name: alarmmanager\alarms

Default Value: False

Recommended Value: False

XML Type: boolean

dotNet Type: System.Boolean

Example:

<ts:section name="alarmmanager">
      ...
    <ts:setting name="alarms" type="boolean" access="visible">
        <ts:value>0</ts:value>
    </ts:setting>
      ...
</ts:section>

Documentation:

Boolean:
0: Alarm management is disabled [Default].
1: Alarm management is enabled in the runtime, i.e. the programming system allows to insert an alarm configuration

PlugIns:

AlarmConfigurationObjects.plugin (3.5.17.0)
VisualEditor.plugin (3.5.17.0)
