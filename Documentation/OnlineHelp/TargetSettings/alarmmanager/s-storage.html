7.3.1.3. storageÂ¶

Name: alarmmanager\storage

Required Runtime Components: CAAStorage

Default Value: False

Recommended Value: False

XML Type: boolean

dotNet Type: System.Boolean

Example:

<ts:section name="alarmmanager">
      ...
    <ts:setting name="storage" type="boolean" access="visible">
        <ts:value>0</ts:value>
    </ts:setting>
      ...
</ts:section>

Documentation:

Boolean:
			0: Persistent storage of alarms is disabled [Default].
			1: Persistent storage of alarms is enabled in the runtime, i.e. the programming system allows to insert an alarm storage object

PlugIn:

AlarmConfigurationObjects.plugin (3.5.17.0)
