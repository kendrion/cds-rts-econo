6.24.1.3. basekeysÂ¶

Name: visualization\keyboardusage\basekeys

Default Value: default

Recommended Value: default

XML Type: string

dotNet Type: System.String

Example:

<ts:section name="visualization">
      ...
    <ts:section name="keyboardusage">
          ...
        <ts:setting name="basekeys" type="string" access="visible">
            <ts:value>default</ts:value>
        </ts:setting>
          ...
    </ts:section>
      ...
</ts:section>

Documentation:

Allows the usage of a predefined set of keys. If no basekeys setting is given, all available keys must be 
defined in the section 'additionalkeys'. 
Possible values: 'default': the following keys are available (given the key, keycode and a description)
'A' to Z'		0x41-0x5A			letter keys
0 to 9			0x30-0x39			number keys
NUM0 to NUM9	0x60-0x69			number keys in number pad
F1 to F12		0x70-0x7B			function keys
BACKSPACE		0x08	
TAB				0x09	
RETURN			0x0D	
PAUSE			0x13	
SPACE			0x20	
END				0x23	
HOME			0x24	
LEFT			0x25	
UP				0x26	
RIGHT			0x27	
DOWN			0x28	
PRINT			0x2A	
INSERT			0x2D	
DELETE			0x2E	
MULTIPLY		0x6A				*
ADDITION		0x6B				+
SUBTRACT		0x6D				-
COMMA			0x6E				,
DIVIDE			0x6F				/

PlugIn:

VisualObject.plugin (3.5.15.0)
