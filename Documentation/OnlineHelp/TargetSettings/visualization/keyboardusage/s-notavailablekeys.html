Name: visualization\keyboardusage\notavailablekeys

XML Type: string

dotNet Type: System.String

Example:

<ts:section name="visualization">
      ...
    <ts:section name="keyboardusage">
          ...
        <ts:setting name="notavailablekeys" type="string" access="visible">
            <ts:value></ts:value>
        </ts:setting>
          ...
    </ts:section>
      ...
</ts:section>

Documentation:

This section allows to remove some of the keys that are previously defined (probably using the "basekeys") 
because they are not available on the current platform. Please see the following example (value is ignored):
	<ts:section name="notavailablekeys">
		<ts:setting name="BACKSPACE" type="string" access="visible">
		<ts:value/>
		</ts:setting>
	</ts:section>

PlugIn:

VisualObject.plugin (3.5.17.0)
