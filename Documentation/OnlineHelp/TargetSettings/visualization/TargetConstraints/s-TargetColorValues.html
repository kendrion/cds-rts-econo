Name: visualization\TargetConstraints\TargetColorValues

XML Type: string

dotNet Type: System.String

Example:

<ts:section name="visualization">
      ...
    <ts:section name="TargetConstraints">
          ...
        <ts:setting name="TargetColorValues" type="string" access="visible">
            <ts:value></ts:value>
        </ts:setting>
          ...
    </ts:section>
      ...
</ts:section>

Documentation:

This setting allows the restriction of colors that are available for the visualization. The value is a string 
consisting of a list of color values (decimal or hexadecimal) in RGB-order specifying an available color each.
The values in this list are separated by the pipe character '|'

PlugIn:

VisualObject.plugin (3.5.17.0)
