XML Type: string

              ...
        </ts:section>
          ...
    </ts:section>
      ...
</ts:section>

Documentation:

A list of visual elements which are not supported by the device. If this list is empty all elements of the active profile
are available.  Only one of the both targetsettings 'AvailableElements' and 'NotAvailableElements' should be used to restrict the number of available elements.

PlugIn:

VisualObject.plugin (3.5.17.0)
