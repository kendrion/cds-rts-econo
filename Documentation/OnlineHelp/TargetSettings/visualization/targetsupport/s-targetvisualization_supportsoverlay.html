Name: visualization\targetsupport\targetvisualization_supportsoverlay

Runtime Version: 3.5.17.0

Compiler Version: 3.5.17.0

Default Value: False

Recommended Value: False

XML Type: boolean

dotNet Type: System.Boolean

Example:

<ts:section name="visualization">
      ...
    <ts:section name="targetsupport">
          ...

Documentation:

Whether the target visualization device supports overlay mode. This
setting can be activated if the overlay feature for the
targetvisualization is supported by the device and compiled into the
runtime system
