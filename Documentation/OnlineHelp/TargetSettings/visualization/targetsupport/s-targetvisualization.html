Name: visualization\targetsupport\targetvisualization

Default Value: True

Recommended Value: True

XML Type: boolean

dotNet Type: System.Boolean

Example:

<ts:section name="visualization">
      ...
    <ts:section name="targetsupport">
          ...
        <ts:setting name="targetvisualization" type="boolean" access="visible">
            <ts:value>1</ts:value>
        </ts:setting>
          ...
    </ts:section>
      ...
</ts:section>

Documentation:

Whether the targetvisualization is supported. This setting has the value true for historic reason so it must be 
disabled explicitly with a value false if this feature is not supported by a device
