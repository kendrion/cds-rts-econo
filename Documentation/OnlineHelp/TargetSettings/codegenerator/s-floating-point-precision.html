Name: codegenerator\floating-point-precision

Compiler Version: 3.5.4.0

XML Type: string

Example:

Documentation:

CortexM3 (ARM Thumb2) code generation: Type of the floating point unit.
                The only supported value is 'single' to use the FPU only for REAL.
                Only evaluated if 'Floating Point Unit' is 'false'
