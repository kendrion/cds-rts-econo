Name: trendrecording\limitvalue

Default Value: 16777216

Recommended Value: 16777216

XML Type: integer

dotNet Type: System.Int32

Example:

<ts:section name="trendrecording">
      ...
    <ts:setting name="limitvalue" type="integer" access="visible">
        <ts:value>16777216</ts:value>
    </ts:setting>
      ...
</ts:section>

Documentation:

Default value for the limit to use according to the currently selected limit type:
0: (Not used).
1: Number of records.
2: Storage size in bytes

PlugIn:

TrendRecordingObjects.plugin (3.5.17.0)
