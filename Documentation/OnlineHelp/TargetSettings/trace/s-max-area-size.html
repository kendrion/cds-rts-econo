Name: trace\max-area-size

Default Value: -1

Recommended Value: -1

XML Type: integer

dotNet Type: System.Int32

Example:

<ts:section name="trace">
      ...
    <ts:setting name="max-area-size" type="integer" access="visible">
        <ts:value>-1</ts:value>
    </ts:setting>
      ...
</ts:section>

Documentation:

The maximum area size for the trace application. Only used if target
setting tracemanager is false.

TraceObject.plugin (3.5.17.0)
