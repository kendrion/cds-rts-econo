start-address¶

Name: memory-layout\areas\area_{0}\start-address

Default Value: 0

Recommended Value: 0

XML Type: integer

dotNet Type: System.Int32

Example:

<ts:section name="memory-layout">
      ...
    <ts:section name="areas">
          ...
        <ts:section name="area_{0}">
              ...
            <ts:setting name="start-address" type="integer" access="visible">
                <ts:value>0</ts:value>
            </ts:setting>
              ...
        </ts:section>
          ...
    </ts:section>
      ...
</ts:section>

Documentation:

Only used for Areas with Flag “FixedSize”!  
The start address for the area as used on the runtime system. If 
the area contains data (variables), any reference in code to this will 
not be relocated on the runtime system, but will contain the correct 
address.

PlugIn:

LanguageModelManager.plugin (3.5.17.0)
