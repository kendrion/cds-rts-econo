Detailed Description¶

Example to handle a run/stop switch on the controller.

USAGE:

  • CH_INIT3: call InitRunStopSwitch() from your component hook function
  • CH_EXIT3: call ExitRunStopSwitch from your component hook function
  • CH_COMM_CYCLE: call CyclicRunStopSwitch() from your component hook function
  • PollRunStopSwitch(): Implement the polling routines for the RunStop Switch state here

static RTS_I32 PollRunStopSwitch (void)¶

Function to get the actual state of the run/stop switch. Is called cyclically from CyclicRunStopSwitch(). Returns:

*0 = switch is in stop state *1 = switch is in run state

static RTS_RESULT InitRunStopSwitch (void)¶

Function to init example at startup of the runtime system. USAGE:

  • Must be called in CH_INIT3 out of component HookFunction()!

Returns:

error code

static RTS_RESULT ExitRunStopSwitch (void)¶

Function to exit example code at shutdown of the runtime system. USAGE:

  • Must be called in CH_EXIT3 out of component HookFunction()!

Returns:

error code

static RTS_RESULT CyclicRunStopSwitch (void)¶

Function to poll cyclically the state of the run/stop switch and to react correctly. USAGE:

  • Must be cyclically. Can be called e.g. from CH_COMM_CYCLE.

Returns:

error code
