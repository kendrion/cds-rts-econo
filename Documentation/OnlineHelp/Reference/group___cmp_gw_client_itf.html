RTS_RESULT GWClientConnectionReady (RTS_HANDLE hDriverHandle, RTS_HANDLE hConnHandle, RTS_I32 nAction)¶

May be called by a commdriver if a connection is ready to send/receive. This function shall not signal a condition more then once without the gateway client reacting to that signal. E.g. if the commdriver signals that a connection is ready to send it shall not signal this situation again before the gateway client actually called “send”. The gwclient must not rely on this function to be called but may use it to increase the performance of the connection.

Parameters

[in]

hDriverHandle

The driver handle returned by GWClientRegisterCommDrv.

[in]

hConnHandle

The handle to the connection.

[in]

nAction

The action that is ready on this connection - eg. COMMDRV_ACTION_SEND, COMMDRV_ACTION_RECEIVE
