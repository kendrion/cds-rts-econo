#define MEM_GET_STATIC_LEN (sizeof( RTS_PCB ) + 2 * MEMPOOL_BLOCK_ALIGNMENT + (Num) * ALIGN_SIZE(sizeof(Struct), MEMPOOL_BLOCK_ALIGNMENT ) + (Num) * sizeof( RTS_BCB ))¶

Macros to handle static size of a MemPool Can be use to calculate the gross size of a static MemPool by specifying the structure of one element. Usage: #define NUM_OF_STATIC_ELEMENTS 10 typedef struct { … } StructNameOfOneElement; static RTS_UI8 s_StaticMemPool[ MEM_GET_STATIC_LEN(NUM_OF_STATIC_ELEMENTS, StructNameOfOneElement) ];
