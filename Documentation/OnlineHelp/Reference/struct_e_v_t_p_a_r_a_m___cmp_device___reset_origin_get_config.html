Field Documentation¶

RTS_HANDLE EVTPARAM_CmpDevice_ResetOriginGetConfig::hUser¶

User handle of the current user.

RTS_UI32 EVTPARAM_CmpDevice_ResetOriginGetConfig::ui32ResetOriginConfigOptions¶

Configuration bit vector representing the possible objects that are handled. See category “Reset Origin Configuration”. The callback has to set the bit for the object it is responsible for.

RTS_UI32 EVTPARAM_CmpDevice_ResetOriginGetConfig::ui32ResetOriginConfigRights¶

Bit vector indicating the access rights for the specified user to delete the objects represented in ui32ResetOriginConfigOptions (bit=0: the current user is allowed to delete the corresponding object, bit=1: the user is not allowed to delete).
