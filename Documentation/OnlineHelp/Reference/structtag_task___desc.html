Detailed Description¶

<SIL2>

Task Description

The different measured jitter terms are defined in RTS Docu: CODESYSControl V3: Manual: Overview of the Kernel Components and Main Functions: Jitter of IEC Tasks.

See: :ref:`ov_kernel_iec_task_jitter`

The measured jitter values J_per_current, J_per_min, and J_per_max are stored in the struct tagTask_Info . All other measured jitter and latency values are stored in this struct tagTask_Desc .
