Field Documentation¶

RTS_IEC_STRING* EVTPARAM_AlarmManagerGetCount::pszApplName¶

The name of the application, which has to process the request.

RTS_IEC_UDINT EVTPARAM_AlarmManagerGetCount::udiCountAlarmGroups¶

The number of alarm group IDs the parameter puiAlarmGroupIDs points to. If no alarm group IDs are specified, this parameter is 0.

RTS_IEC_UDINT* EVTPARAM_AlarmManagerGetCount::pudiAlarmGroupIDs¶

Points to an array of UDINT values with the IDs of the alarm groups to use as filter. If no such filter has been specified this parameter is NULL.

RTS_IEC_UDINT EVTPARAM_AlarmManagerGetCount::udiCountAlarmClasses¶

The number of alarm class IDs the parameter puiAlarmClassIDs points to. If no alarm class IDs are specified, this parameter is 0.

RTS_IEC_UDINT* EVTPARAM_AlarmManagerGetCount::pudiAlarmClassIDs¶

Points to an array of UDINT values with the IDs of the alarm classes to use as filter. If no such filter has been specified this parameter is NULL.

RTS_IEC_USINT EVTPARAM_AlarmManagerGetCount::usiPriorityFrom¶

A byte value with minimum priority the alarm must have.

RTS_IEC_USINT EVTPARAM_AlarmManagerGetCount::usiPriorityTo¶

A byte value with maximum priority the alarm must have.

RTS_IEC_UDINT EVTPARAM_AlarmManagerGetCount::udiCount¶

The total number of relevant alarms

RTS_IEC_BOOL EVTPARAM_AlarmManagerGetCount::bRequestHandled¶

Returns TRUE if the request has been handled, otherwise FALSE
