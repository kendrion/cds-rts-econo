7.1.5.692. SERVICEHANDLER_PARAMETER Struct Reference¶

Detailed Description¶

Return values

ERR_OK

Received service was completely handled.

ERR_L7_UNKNOWNCMD

Service is unknown by the event handler.

ERR_CALL_AGAIN

Received service will be handled asynchronously. To progress this event will be posted again for the same received service data with bFirstCall=0.

ERR_…

Another error occurred, caller will send this result in a generic error tag as reply (or close the channel).
