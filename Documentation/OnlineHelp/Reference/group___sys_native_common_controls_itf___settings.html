#define SYSNATIVECOMMONCONTROLS_KEY_PLUGINS_ENABLED “PluginsEnabled”¶

Setting defining whether on certain platforms (e.g. Linux/QT) plugins displaying special web content are enabled (value 1) or not (value 0). For security reasons plugins are disabled by default.
