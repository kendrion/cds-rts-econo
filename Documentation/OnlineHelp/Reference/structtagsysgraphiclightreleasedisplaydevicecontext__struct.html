5.3.1882.2. Detailed Description¶

Releases a handle to the basic display context that was returned by SysGraphicLightGetDisplayDeviceContext.

Parameters

[in] hDisplayContext The display device context to release.

Returns:

*RESULT: Returns the runtime system error code (see CmpErrors.library).
