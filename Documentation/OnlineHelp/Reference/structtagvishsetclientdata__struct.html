Detailed Description¶

Registers the IEC-ID of the client and the paint-buffer of this client | in the runtime system. | A id of -1 means that the client was not registered! | A dwExternID of 0xFFFFFFFE, i.e. REMOVE_CLIENT means that the client with IEC_ID id | has to be removed from the maps in the runtime. The data in the IEC-Task has already be | freed! link even in simulation mode
