5.2.76.9.1. RTS_HANDLE ObjManDictionaryOpen (CMPID CmpId, RTS_SIZE capacity, RTS_BOOL bRequireSync)¶

Opens an object dictionary for the component with CmpId.

This routine should be called during one of the early phases of system initialization process.

One component can open a single dictionary only.

Parameters

[in] CmpId Specifies a component ID
[in] capacity Supplies the size for the dictionary. The more capacity, the better performance..
[in] bRequireSync if TRUE, sets a requirement to create a thread-safe dictionary for this component

Returns:

Returns a handle to an object dictionary

5.2.76.9.2. RTS_RESULT ObjManDictionaryClose (RTS_HANDLE hDictionary)¶

Closes an object dictionary.

This routine should be called during one of the late phases of system shutdown process.

Parameters

[in] hDictionary Supplies a handle to a dictionary that has to be closed

Returns:

Error code:

  • ERR_OK: Dictionary has been closed successfully
  • ERR_PARAMETER: Invalid handle passed as the routine parameter
  • ERR_FAILED: Failed to close a dictionary

5.2.76.9.3. IKKey ObjManDictionaryAddObject (RTS_HANDLE hDictionary, RTS_HANDLE hObject)¶

Adds an object to a dictionary returning a 32-bit key.

Parameters

[in] hDictionary Supplies a handle to a dictionary
[in] hObject Supplies a handle to an object that has to be added to a dictionary

Returns:

Returns a key to an object, or IKHS_NO_KEY in case of failure

5.2.76.9.4. RTS_RESULT ObjManDictionaryRemoveObject (RTS_HANDLE hDictionary, RTS_HANDLE hObject)¶

Removes an object from a dictionary.

Parameters

[in] hDictionary Supplies a handle to a dictionary
[in] hObject Supplies a handle to an object that has to be removed from a dictionary

Returns:

Error code:

  • ERR_OK: Object has been removed successfully
  • ERR_PARAMETER: Invalid handle passed as the routine parameter
  • ERR_FAILED: Object not found

5.2.76.9.5. RTS_RESULT ObjManDictionaryRemoveKey (RTS_HANDLE hDictionary, IKKey key)¶

Removes an object from a dictionary by a key.

Parameters

[in] hDictionary Supplies a handle to a dictionary
[in] key Supplies a key to an object that has to be removed from a dictionary

Returns:

Error code:

  • ERR_OK: Object has been removed successfully
  • ERR_PARAMETER: Invalid handle or key passed as the routine parameter
  • ERR_FAILED: Object not found

5.2.76.9.7. RTS_RESULT ObjManDictionaryGetSize (RTS_HANDLE hDictionary, RTS_SIZE *pSize)¶

Returns the number of elements in a dictionary.

Parameters

[in] hDictionary Supplies a handle to a dictionary
[out] pSize Contains the number of elements in a dictionary

Returns:

Error code:

  • ERR_OK: pSize contains some valid value
  • ERR_PARAMETER: Invalid handle or null pointer passed as the routine parameter
  • ERR_FAILED: Dictionary not found
