Function Documentation¶

RTS_HANDLE ObjManDictionaryOpen (CMPID CmpId, RTS_SIZE capacity, RTS_BOOL bRequireSync)¶

Opens an object dictionary for the component with CmpId.

This routine should be called during one of the early phases of system initialization process.

One component can open a single dictionary only.

Parameters

[in]

CmpId

Specifies a component ID

[in]

capacity

Supplies the size for the dictionary. The more capacity, the better performance..

[in]

bRequireSync

if TRUE, sets a requirement to create a thread-safe dictionary for this component

Returns:

Returns a handle to an object dictionary

RTS_RESULT ObjManDictionaryClose (RTS_HANDLE hDictionary)¶

Closes an object dictionary.

This routine should be called during one of the late phases of system shutdown process.

Parameters

[in]

hDictionary

Supplies a handle to a dictionary that has to be closed

Returns:

Error code:

  • ERR_OK: Dictionary has been closed successfully

  • ERR_PARAMETER: Invalid handle passed as the routine parameter

  • ERR_FAILED: Failed to close a dictionary

Parameters

[in]

Returns:
