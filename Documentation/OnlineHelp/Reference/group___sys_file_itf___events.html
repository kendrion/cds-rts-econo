#define EVT_SysFileOpen MAKE_EVENTID ( EVTCLASS_INFO , 1)¶

Event is sent when a file is opened. It is sent directly after the file was opened. Note: The event is set only with SysFile, it is not set with SysFileFlash. Note: Event set after CH_INIT2. There is no notification of opening a file before this hook.

Parameters

[in]

pEventParam

EVTPARAM_SysFileOpen

#define EVT_SysFileClose MAKE_EVENTID ( EVTCLASS_INFO , 2)¶

Event is sent when a file is closed. It is send directly before the file is closed (file handle still valid). Note: The event is set only with SysFile, it is not set with SysFileFlash. Note: Event set after CH_INIT2. There is no notification of opening a file before this hook.

Parameters

[in]

pEventParam

EVTPARAM_SysFileClose

#define EVT_SysFileFlashGetFileMapIndex MAKE_EVENTID ( EVTCLASS_INFO , 3)¶

The event is sent if the function GetFileMapIndex is called NOTE: This event is fired for every call and every loop call of count of files
