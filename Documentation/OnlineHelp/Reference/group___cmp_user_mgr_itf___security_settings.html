#define CMPUSERMGR_KEY_STRING_LOGIN_AUTH_TYPE CMPSETTINGS_CATEGORY_SECURITY “.UserLogin_AuthenticationType”¶

Security setting to select the authentication algorithm. Can be selected in CmpSecurityManager. NOTE: Security modes must be ordered in an descending sorting, i.e. the setting with the highest security level is found at the beginning of the list!
