#define CMPUSERMGR_KEY_STRING_LOGIN CMPSETTINGS_CATEGORY_SECURITY ”.UserLogin_ErrorBehavior”¶

Security setting of the behavior, if login authentication fails. Can be selected in CmpSecurityManager. NOTE: Security modes must be ordered in an descending sorting, i.e. the setting with the highest security level is found at the beginning of the list!
