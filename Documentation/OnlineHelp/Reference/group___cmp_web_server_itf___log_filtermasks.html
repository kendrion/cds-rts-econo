Macro Definition Documentation¶

#define LFM_WEBSERVER_MASK 0x000000FF¶

Mask to filter debug outputs module or scope specific

#define LFM_WEBSERVER_COMMON 0x00000001¶

Shows messages of the log level ‘info’ (default). Mostly: file not found messages.

#define LFM_WEBSERVER_DEBUG 0x00000010¶

Shows messages of the log level ‘debug’. All messages including not modified and successfully requested.
