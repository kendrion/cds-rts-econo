Macro Definition Documentation¶

#define BLKDRVCOM_MAX_SER_READSIZE 1¶

Maximum number of bytes that are read with one call to SysComRead. Can be set to a bigger value on embedded systems, to minimize the number of calls to SysComRead. Example: 256. Only for BlkDrvCom.

#define MAX_UDP_DEVICES 8¶

Max number of devices (interfaces), which are used by the CmpBlkDrvUdp. New setting BLKDRVUDPKEY_INT_MAX_INTERFACES_DEFAULT should be used instead.

#define UDP_BIND_ADDRESS 0¶

Socket bind option for the CmpBlkDrvUdp.

#define UDP_BIND_INADDR_ANY 1¶

#define UDP_BIND_TWO_SOCKETS 2¶

#define UDP_PACKET_SORT_NONE 0¶

Options for the CmpBlkDrvUdp to sort packages after receiving or before sending. Sorting for incoming and outgoing packages can be used seperately or in combination.
