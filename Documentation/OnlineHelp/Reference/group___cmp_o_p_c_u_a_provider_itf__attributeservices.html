STATICITF_DEF OpcUa_StatusCode OpcUaProviderRead (RTS_HANDLE hProvider, OpcUaProvider_ReadContext *pContext)¶

This function is used to read attributes of the specified node.

If the provider is the owner of the specified node it should fill up the value in the ReadContext. If the node is not known to the provider the provider MUST return OpcUa_BadNodeIdUnknown.

Parameters

[in]

pContext

Information of item to read.

Returns:

Function returns OpcUa_Good if everything went fine. Return OpcUa_Bad if an error occurs.

STATICITF_DEF OpcUa_StatusCode OpcUaProviderWrite (RTS_HANDLE hProvider, OpcUaProvider_WriteContext *pContext)¶

This function is used to write to a specific node item

If the provider is the owner of the specific node it should write the value to the node if possible. If the node is not known to the provider the provider MUST return OpcUa_BadNodeIdUnknown.

Parameters

[in]

pContext

Information of node to write.

Returns:

Function returns OpcUa_Good if everything went fine. Return OpcUa_Bad if an error occurs.
