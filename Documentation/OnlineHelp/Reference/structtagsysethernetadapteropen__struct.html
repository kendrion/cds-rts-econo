5.3.1807.2. Detailed Description¶

Open an ethernet adapter by its MAC address. .. note:: Each adapter can only be opened once. :return: Returns a handle to the opened adapter. SysTypes.RTS_INVALID_HANDLE is returned if the operation failed.
