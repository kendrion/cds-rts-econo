#define CMPX509CERT_KEY_STRING_DIFFIEHELLMANPARAMETER CMPSETTINGS_CATEGORY_SECURITY ”.DiffieHellmanParameter_Generate”¶

Security settings of the X.509 certificate management to create diffie hellman parameters. Can be selected in CmpSecurityManager. NOTE: Security modes must be ordered in an descending sorting, that the setting with the highest security level is found at the beginning of the list!
