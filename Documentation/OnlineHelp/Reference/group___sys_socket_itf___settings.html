Macro Definition Documentation¶

#define SYSSOCKET_ADAPTER_PREFIX “Adapter”¶

Settings prefix for network adapter list. Has to be used for all list settings. The index number, separated by dots, have to be increased by one for each configured adapter name. Adapter names can be configured by an ASCII name oder by an unicode name. If for a index number was found a “NameUnicode” setting, the “Name” setting is ignored. Example: Adapter.0.Name=Lan1 Adapter.1.Name=Lan2 Adapter.2.NameUnicode=”L00a00n00300”

#define SYSSOCKETKEY_WSTRING_NAME_UNICODE “NameUnicode”¶

Setting for network. adapter list. Adapter name in Unicode. .Example: Adapter.0.NameUnicode=”L00a00n00100”

#define SYSSOCKETKEY_STRING_NAME “Name”¶

Setting for network adapter list. Adapter name in ASCII. Only evaluted, if no “NameUnicode” setting for the same adapter index exists. Example: Adapter.0.Name=”Lan1”

#define SYSSOCKETKEY_INT_ENABLE_SETIPANDMASK “EnableSetIpAndMask”¶

Setting for network adapter list to specify, if it is allowed to set the ip address and subnet mask for this adapter. If enabled (1) the ip address and subnet mask can be set by SysSockSetIpAddressAndNetMask() . Example: Adapter.0.Name=”Lan1” Adapter.0.EnableSetIpAndMask=1

#define SYSSOCKETVALUE_INT_ENABLE_SETIPANDMASK_DEFAULT 0¶

#define SYSSOCKETKEY_STRING_QNX_ITFNAME “QNX.ItfName”¶

Setting to specify the socket interface name to be used by the runtime. Only for QNX. The default is “tsec0”.

#define SYSSOCKETKEY_STRING_IPADDRESS “ipaddress”¶

#define SYSSOCKETKEY_STRING_SUBNETMASK “subnetmask”¶

Setting to specify the subnet mask of the target. See setting ipaddress for details.

#define SYSSOCKETKEY_STRING_GATEWAY “gateway”¶

Setting to specify the default gateway of the target. See setting ipaddress for details.

#define SYSSOCKETKEY_STRING_SYSLOG_IPADDRESS “sysLog.IPAddress”¶

Setting to specify the send IP address (destination) of the syslog messages. NOTE: “255.255.255.255” is the broadcast address to send the syslog message to all network members!

#define SYSSOCKETVALUE_STRING_SYSLOG_IPADDRESS_DEFAULT “127.0.0.1”¶

#define SYSSOCKETKEY_INT_SYSLOG_PORT “sysLog.Port”¶

Setting to specify the UDP send port of the syslog messages

#define SYSSOCKETVALUE_INT_SYSLOG_PORT_DEFAULT 514¶
