#define EVT_ScheduleTaskGap MAKE_EVENTID ( EVTCLASS_INFO , 6)¶

Event is sent always, if no IEC task is active (task gap)

Parameters

[in] pEventParam EVTPARAM_CmpScheduleTick

#define EVT_ScheduleTaskGapApp MAKE_EVENTID ( EVTCLASS_INFO , 7)¶

Event is sent always for each application, if no IEC task in that application is active (task gap). The synchronization for the application has to be done in the callback of the event.

Parameters

[in] pEventParam EVTPARAM_CmpScheduleTaskGapApp
