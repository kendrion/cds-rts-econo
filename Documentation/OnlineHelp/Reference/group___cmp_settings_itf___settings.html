#define CMPSETTINGS_KEY_INT_IS_WRITEPROTECTED “IsWriteProtected”¶

Optional setting to specify, if the main config file is writeprotected. 1: writeprotected, file attributes are not checked 0: file attributes are checked (default)

#define CMPSETTINGS_KEY_INT_IS_WRITEPROTECTED_DEFAULT 0¶

#define CMPSETTINGS_KEY_STRING_FILE_REFERENCE “FileReference”¶

Setting to specify external configuration files: File reference list must be indexed, started with 0 or 1, e.g.: FileReference.0=ExtraConfigurationfile.cfg FileReference.1=/tmp/ExtraConfigurationfile2.cfg

The filenames must be unique, but a path may be added. If the file does not exist, the file is created. You can specify sections, that should be created in this referenced file, by a comma separated list. e.g.: FileReference.0=ExtraConfigurationfile.cfg, CmpApp, CmpDevice

There is an optional atrribute to writeprotect an extra configuration file (the same effect as the setting for the main file, see above): FileReference.0.IsWriteProtected=1
