Macro Definition Documentation¶

#define RTS_CMINIT_OPTION_SETTINGSFILEISOPTIONAL UINT32_C(0x00000001)¶

Specifies, if the settings file is only used optional

#define RTS_CMINIT_OPTION_DONTCALLINITHOOKS UINT32_C(0x00000002)¶

If this option is set, the CMInit sequence is stopped right after CH_INIT hook. To finish the startup sequence, you have to call CMInitEnd() afterwards. This can be used, if you have to do things first before all other components, but you need here the runtime system components.

#define RTS_CMINIT_OPTION_INITTASK UINT32_C(0x00000004)¶

Call CH_INIT_XXX hooks out of a SysTask

#define RTS_CMINIT_OPTION_COMMCYCLETASK UINT32_C(0x00000008)¶

Call CH_COMM_CYCLE out of a SysTask

#define RTS_CMINIT_OPTION_EXITTASK UINT32_C(0x00000010)¶

Call CH_EXIT_XXX hooks out of a SysTask
