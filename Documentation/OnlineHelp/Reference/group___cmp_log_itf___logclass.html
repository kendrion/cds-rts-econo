Macro Definition Documentation¶

#define LOG_NONE INT32_C(0x00000000)¶

Log entry classes and filters No defined class

#define LOG_INFO INT32_C(0x00000001)¶

Information

#define LOG_WARNING INT32_C(0x00000002)¶

Warning

#define LOG_ERROR INT32_C(0x00000004)¶

Error

#define LOG_EXCEPTION INT32_C(0x00000008)¶

Exception

#define LOG_DEBUG INT32_C(0x00000010)¶

Log entries for internal debugging (reserved for development, diagnosis or test)

#define LOG_PRINTF INT32_C(0x00000020)¶

Debug log entries like printf. For this entry, no timestamp is logged!

#define LOG_COM INT32_C(0x00000040)¶

Log entries for communication stuff. Should be separated in own logger!

#define LOG_INFO_TIMESTAMP_RELATIVE INT32_C(0x00000080)¶

Information with timestamp relative to the previous entry

#define LOG_CRIT_SEC INT32_C(0x00000100)¶

Semaphore log entries for testing (deadlocks, etc.)

#define LOG_USER_NOTIFY INT32_C(0x00010000)¶

Log entry is displayed as a message box in the development system!
