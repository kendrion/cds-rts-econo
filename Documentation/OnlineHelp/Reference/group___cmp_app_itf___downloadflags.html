Macro Definition Documentation¶

#define DLF_CONTINUEDOWNLOAD UINT32_C(0x00000001)¶

Download flags that are transmitted with each download. Continue download, if download is transmitted in several blocks

#define DLF_USEBOOTAPPFROMDOWNLOAD UINT32_C(0x00000002)¶

This flag indicates newer PS where the user can define whether a boot project shall be written (to switch on next flag)

#define DLF_WRITEBOOTAPPLICATION UINT32_C(0x00000004)¶

This flag indicates that a the boot application shall be written with download
