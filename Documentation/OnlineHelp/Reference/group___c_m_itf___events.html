#define EVT_CmpMgr_DisableOperation MAKE_EVENTID ( EVTCLASS_INFO , 6)¶

Event is sent to disable operations. Each operation is defined in the corresponding Itf.h file of the component, which is specified by its ComponentID

Parameters

[in] pEventParam EVTPARAM_CmpMgr_DisableOperation

#define EVT_CmpMgr_PrepareExitCommProcessing MAKE_EVENTID ( EVTCLASS_INFO , 7)¶

Event is sent after EVT_CmpMgr_PrepareExitComm and can be used if comm cycles are necessary to perform some kind of shutdown operation. The event may be called several times if one of the callees requests further calls. This can happen if the shutdown operation takes some time.

Parameters

[in] pEventParam EVTPARAM_CmpMgr_Shutdown

#define EVT_CmpMgr_LicenseRequest MAKE_EVENTID ( EVTCLASS_INFO , 8)¶

Event to get informed when a license was requrested. Only for informational use. No result of the event will be evaulated.

Parameters
