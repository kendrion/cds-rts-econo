#define CMPAPP_KEY_STRING_CODEPROTECTION CMPSETTINGS_CATEGORY_SECURITY ”.CodeProtection”¶

Security settings of the application downloaded code and bootproject. NOTE: Security modes must be ordered in an descending sorting, that the setting with the highest security level is found at the beginning of the list!
