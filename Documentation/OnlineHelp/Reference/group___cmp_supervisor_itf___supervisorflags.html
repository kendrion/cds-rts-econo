Macro Definition Documentation¶

#define RTS_SUPERVISOR_FLAG_NONE 0¶

No flag specified

#define RTS_SUPERVISOR_FLAG_RESET_NECESSARY 0x00000001¶

Operation failed and so a reset is necessary!

#define RTS_SUPERVISOR_FLAG_WATCHDOG 0x00010000¶

This is a watchdog (consumer), that uses the supervisor to protect the controller! NOTE: This consumer can be disabled via the standard function SupervisorOperationDisable() and so this watchdog will not expire, if any of the supervisor operations will fail!
