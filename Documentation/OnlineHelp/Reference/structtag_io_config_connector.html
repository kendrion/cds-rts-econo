5.3.1274.2. Detailed Description¶

<SIL2>

Connector information. Each device is described completely as a set of one input- and one or more output-conenctors.

Parameters

[in] wType Type of the connector. See category “Connector types” for detailed information.
[in] wOptions Connector options. See category “Connector options” for detailed information.
[in] dwFlags Connector flags. See category “Connector flags” for detailed information.
[in] hIoDrv Registered instance handle of the IO-driver, that operates this device respectively this connector
[in] dwNumOfParameters Number of parameters in the parameter list
[in] pParameterList Pointer to the parameter list, that contains all parameters of this connector
[in] pFather Pointer to the father connector to describe the device tree
