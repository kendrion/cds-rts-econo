#define EVT_PlcShellCommand MAKE_EVENTID ( EVTCLASS_INFO , 1)¶

This event is sent to all command handlers. It has to be handled only if the command name matches.

Parameters

[in] pEventParam EVTPARAM_PlcShellCommand

#define EVT_PlcShellCommandRegister MAKE_EVENTID ( EVTCLASS_INFO , 2)¶

This event is sent in CH_INIT_TASKS for each registered PlcShell command. It can be used to disable the command by setting the bDisable flag (see EVTPARAM_PlcShellCommandRegister ).

Parameters

[in] pEventParam EVTPARAM_PlcShellCommandRegister
