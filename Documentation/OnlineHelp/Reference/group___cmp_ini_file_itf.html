Function Documentation¶

RTS_HANDLE IniOpen (const char *pszFileName, RTS_UI32 options, RTS_RESULT *pResult)¶

Function to open an ini-file

Parameters

[in]

pszFileName

Name and path of the ini-file

[in]

options

Options to open ini-file. See category “Ini-file options”

[out]

pResult

Error code

Return values

ERR_OK

Ini file can be opened

ERR_NO_OBJECT

Ini file cannot be opened (not existing, see INI_OPTION_NONE)

ERR_FAILED

Ini file cannot be created (see INI_OPTION_CREATE or INI_OPTION_OVERWRITE_EXISTING)

Returns:

Handle to the opened ini-file or RTS_INVALID_HANDLE if failed

RTS_RESULT IniSetFilter (RTS_HANDLE hIniFile, char *pszFilter)¶

Function to set a filter/section list for the ini file! If such a filter list is specified, only these sections can be stored in the ini file!

Parameters

[in]

hIniFile

Handle to the ini file

[in]

pszFilter

List of sections with a comma separated list: Example: “[CmpApp], [CmpDevice]”

Returns:

error code

Return values

ERR_OK

Ini file filter can be set

ERR_PARAMETER

Invalid parameter
