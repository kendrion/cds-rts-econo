Service group: SG_CMPCODEMETER

Service: SRV_CODEMETER_READ_DONGLE_LIST

Response tags:

  • Tag Number of dongles (1..?) (RTS_UI16): Number of dongles in the reply

  • TAG_CM_CMPLX_DONGLE (1..?): Complex tag for dongle entry

    • Tag Dongle handle (1..?) (char*): Dongle handle. Here we use the serial number as string

    • TAG_CM_DONGLE_INFOSTRING (1..?) (char*): Infos string of the dongle
