Function Documentation¶

STATICITF_DEF RTS_HANDLE OpcUaProviderCreate (RTS_HANDLE hIOpcUaProvider, CLASSID ClassId, int iId, RTS_RESULT *pResult)¶

Create a new OPC UA provider instance. Is typically internally used by the provider itself

Parameters

[in]

hIOpcUaProvider

Handle to the provider interface. Must be 0 and is filled automatically by calling the CAL_OpcUaProviderCreate() macro!

[in]

ClassId

ClassID of the driver. See “Class IDs” section in CmpItf.h or in the Dep.h file of the provider.

[in]

iId

Instance number of the provider

[out]

pResult

Pointer to error code

Returns:

Should return RTS_INVALID_HANDLE

STATICITF_DEF RTS_RESULT OpcUaProviderDelete (RTS_HANDLE hOpcUaProvider)¶

Delete an OPC UA provider instance. Is typically internally used by the provider itself

Parameters

[in]

hOpcUaProvider

Handle to the provider instance

Returns:

Should return ERR_NOTIMPLEMENTED

STATICITF_DEF OpcUa_StatusCode OpcUaProviderInitialize (RTS_HANDLE hProvider)¶

This function is called by the OPC UA server to initialize the provider. After the call to this function the provider should be ready to be used. Returns:

Function returns OpcUa_Good if everything went fine. Return OpcUa_Bad if an error occurs.

STATICITF_DEF OpcUa_StatusCode OpcUaProviderCleanup (RTS_HANDLE hProvider)¶

This function is called by the OPC UA server to cleanup the provider. Resources should be clean up. Returns:

Function returns OpcUa_Good if everything went fine. Return OpcUa_Bad if an error occurs.
