5.3.45.3. Field Documentation¶

5.3.45.3.1. OpcUa_Boolean _OpcUaVariableDescription::bValueAvailable¶

Static value of the node. Flag if the value is set or not:

  • If this value is OpcUa_True the read operation will be done by the server. This is useful for static values. Write access will be denied.
  • If this value is OpcUa_False the OPC UA Server will call OpcUaProviderRead of the provider that has added this node. OpcUaProviderWrite to write this node.

5.3.45.3.3. OpcUa_NodeId _OpcUaVariableDescription::dataType¶

5.3.45.3.4. OpcUa_Int32 _OpcUaVariableDescription::i32ValueRank¶

ValueRank of the value. Mandatory.

5.3.45.3.5. OpcUa_Byte _OpcUaVariableDescription::accessLevel¶

Access level of the node. If bValueAvailable == OpcUa_True this will be changed to CurrentRead. Mandatory.

5.3.45.3.6. OpcUa_Byte _OpcUaVariableDescription::useraccesslevel¶

Access level of the node with user rights taken into account. If pszObject is set the CmpUserMgr will be called to get the rights. Mandatory.

5.3.45.3.7. OpcUa_Boolean _OpcUaVariableDescription::historizing¶

Flag if historizing is available. By now not supported by the OPC UA server. Mandatory.

5.3.45.3.8. OpcUa_Boolean _OpcUaVariableDescription::bArrayDimensionSet¶

Flag if the array dimension is set.

5.3.45.3.9. OpcUa_Variant _OpcUaVariableDescription::arrayDimensions¶

Description of the array dimensions. Mandatory if the ValueRank is not Scalar.

5.3.45.3.10. OpcUa_Boolean _OpcUaVariableDescription::bMinimumSamplingIntervalSet¶

Flag if the minimum sampling interval is set.
