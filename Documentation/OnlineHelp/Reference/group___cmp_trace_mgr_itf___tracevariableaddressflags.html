#define TRACE_VAR_ADDRESS_FLAGS_IEC 0x00000001¶

Trace variable address flags.

Usage: The flags mus be used in combination. Here are some typical combinations: ulFlags = TRACE_VAR_ADDRESS_FLAGS_IEC | TRACE_VAR_ADDRESS_FLAGS_AREA_OFFSET; ulFlags = TRACE_VAR_ADDRESS_FLAGS_IEC | TRACE_VAR_ADDRESS_FLAGS_PROPERTY; ulFlags = TRACE_VAR_ADDRESS_FLAGS_IEC | TRACE_VAR_ADDRESS_FLAGS_POINTER; ulFlags = TRACE_VAR_ADDRESS_FLAGS_IEC | TRACE_VAR_ADDRESS_FLAGS_MONITORING; ulFlags = TRACE_VAR_ADDRESS_FLAGS_IOCONFIG; ulFlags = TRACE_VAR_ADDRESS_FLAGS_SYSTEM; IEC variable (automatic update)

#define TRACE_VAR_ADDRESS_FLAGS_IOCONFIG 0x00000002¶

IO-Config variable (automatic update)

#define TRACE_VAR_ADDRESS_FLAGS_SYSTEM 0x00000004¶

System variable (update must do the system or IEC program!)

#define TRACE_VAR_ADDRESS_FLAGS_POINTER 0x00000010¶

Address specified by a pointer

#define TRACE_VAR_ADDRESS_FLAGS_AREA_OFFSET 0x00000020¶

Address specified by area/offset (IEC)

#define TRACE_VAR_ADDRESS_FLAGS_PROPERTY 0x00000040¶

Address is a property variable (IEC)

#define TRACE_VAR_ADDRESS_FLAGS_MONITORING 0x00000080¶

Address is an item with a monitoring request (IEC)

#define TRACE_VAR_ADDRESS_FLAGS_SYMBOLIC 0x00000100¶

The variable will be accessed using the symbol configuration based on its name

#define TRACE_VAR_ADDRESS_FLAGS_MONITORING2 0x00000200¶

Address is an item with a monitoring request for CmpMonitor2 (IEC)
