Macro Definition Documentation¶

#define USERDB_RIGHT_NONE UINT32_C(0x00000000)¶

Defined rights that can fully control the access to an object Access denied

#define USERDB_RIGHT_VIEW UINT32_C(0x00000001)¶

Only view rights on the object. Object cannot be modified in any way!

#define USERDB_RIGHT_MODIFY UINT32_C(0x00000002)¶

Object can be viewed and modified

#define USERDB_RIGHT_EXECUTE UINT32_C(0x00000004)¶

Object can be executed

#define USERDB_RIGHT_ADD_REMOVE UINT32_C(0x00000008)¶

It is allowed to add and remove sub objects

#define USERDB_RIGHT_OEM UINT32_C(0xFF000000)¶

The upper 8 bits of the access rights are reserved for OEM specific access rights

#define USERDB_RIGHT_ALL UINT32_C(0xFFFFFFFF)¶

All rights allowed on the object
