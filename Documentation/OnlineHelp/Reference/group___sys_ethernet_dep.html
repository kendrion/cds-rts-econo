Detailed Description¶

A driver for ethercat (udp) networks. It is able to scan a connected ethercat ring for slaves Detected network interfaces are named “ether 0” through “ether n” where n+1 is the number of detected network interfaces.

Macro Definition Documentation¶

#define SYS_ETHERNET_MIN_NUM_FRAMES 100¶

Defines the minum number of ethernet frames to be available. This number is statically available.

#define SYS_ETHERNET_MAX_NUM_FAMES RTS_UI32_MAX¶

The maximum number of frames that should be available on the system. No more fames will be allocated from the heap if this number is reached. If this number matches the SYS_ETHERNET_MIN_NUM_FRAMES no fames will be allocated at all.
