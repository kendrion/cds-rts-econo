Setting to define the trace level of the cifx toolkit.

#define CMPHILSCHERCIFX_INTERRUPT_MODE “InterruptMode”¶

<type>Int</type>

Setting to define, if the NetX is used interrupt driven. SysInt must be supported in this case!

#define CMPHILSCHERCIFX_INT_INTERRUPT_MODE_DEFAULT 0¶

#define CMPHILSCHERCIFX_DMA_MODE “DMAMode”¶

<type>Int</type>

Setting to define, if the NetX is using DMA mode. NOTE:

  • Compiler Switch CIFX_TOOLKIT_DMA must be set additionally (see above)!

  • In OS adaptation layer: OS_MemAllocDMABuffer and OS_MemFreeDMABuffer must be supported!

  • If DMA mode is off, the IoDrvCifxProfibus uses a buffered memory to update shared memory instead of DMA.

#define CMPHILSCHERCIFX_INT_DMA_MODE_DEFAULT 0¶

#define CMPHILSCHERCIFX_STRING_BOOTLOADER_FILE “BootloaderFile”¶

<type>String</type>

Setting to define the boot loader file.

Example: BootloaderFile=NETX100-BSL.bin or Device.0.BootloaderFile=NETX100-BSL.bin

Setting to enable dynamic firmware loading matching to IO-configuration out of CODESYS.

#define CMPHILSCHERCIFX_KEY_INT_DPM_ADDRESS “DPMAddress”¶

<type>Int</type>

Setting to define the DPM address of the NetX device. The address is the physical address of the DPM or if necessary the still mapped address. NOTE: The setting must be specified with the device prefix and number.

Example: Device.0.DPMAddress=0x12345678
