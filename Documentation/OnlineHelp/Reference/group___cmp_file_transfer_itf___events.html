EVTPARAM_CmpFileTransfer

#define EVT_SyncFileTransfer MAKE_EVENTID ( EVTCLASS_INFO , 2)¶

This event is sent before and after synchronized files are downloaded. Typically this occurs after an application download or online change.

[in]

pEventParam
