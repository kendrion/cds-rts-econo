#define SRV_READ_STATUS 0x14¶

Service group: SG_APPLICATION

Service: SRV_READ_STATUS

Request tags:

  • Tag Application description (1..?)

    • Tag Task index (0..?) (RTS_UI32): Task index

    • Tag ID (1..?) (RTS_UI32): Session ID; triggers READ_STATUS_RESULT, READ_STATUS_EXECUTIONPOSITION, READ_STATUS_BPHITCOUNT, READ_STATUS_BPHITCOUNTEX

    • Tag Version (0..?): triggers READ_STATUS_INSTANCEPOSITION, READ_STATUS_DEBUGTASK

    • Tag User notify (0..?): triggers READ_USER_NOTIFY

Response tags:

  • Tag Read status reply (1..?): Top level tag may contain the following sub tags

    • Tag Result (0..?) (RTS_UI16, RTS_UI32, RTS_UI32): Result code, App state, App OpState

    • Tag Execution position (0..?) (RTS_UI16, RTS_UI32): Area, Offset of callstackEntry

    • Tag Data BP position (0..?) (RTS_UI16, RTS_UI32): Area, Offset of reached data breakpoint

    • Tag BP hit count (0..?) (RTS_UI32): HitCount

    • Tag BP hit count ex (0..?) (RTS_UI16, RTS_UI16, RTS_UI32, RTS_UI32): Area, state, Offset, current hitcount of BP

    • Tag Instance position (0..?) (RTS_UI16, RTS_UI32): AreaInstance, OffsetInstance of callstackEntry

    • Tag Debug task (0..?) (RTS_UI16): DebugTaskIdx

    • READ_USER_NOTIFY (0..?) (…): Response of SrvGetUserNotificationService2

    • Tag Last change

  • Tag Operation mode (0..?) (RTS_UI32): Device operation mode

#define SRV_READ_CALLSTACK 0x16¶

Service group: SG_APPLICATION

Service: SRV_READ_CALLSTACK

Request tags:

  • TAG_APPL_APPLICATIONDESC (1..?)

    • TAG_APPL_TASK_INDEX (0..?) (RTS_UI32): Task index

    • TAG_APPL_ID (1..?) (RTS_UI32): Session ID

Response tags:

  • READ_STATUS_REPLY (1..?): Top level tag may contain the following sub tags

    • READ_STATUS_STACKPOINTER (0..?) (RTS_UINTPTR): Stackpointer

    • READ_STATUS_EXECUTIONPOSITION (0..?) (RTS_UI16, RTS_UI32): Area, Offset of callstackEntry

    • READ_STATUS_INSTANCEPOSITION (0..?) (RTS_UI16, RTS_UI32): AreaInstance, OffsetInstance of callstackEntry
