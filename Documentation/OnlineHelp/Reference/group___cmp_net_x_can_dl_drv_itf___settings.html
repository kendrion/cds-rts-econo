#define NETXCMD_KEY_MAX_NETWORKS “MaxNumberOfCanNetworks”¶

<type>Int</type>

Maximal number of NetX CAN networks, supported by this device. This setting is mandatory

#define NETXCMD_INT_MAX_NETWORDS_DEFAULT 1¶

#define NETXCMD_KEY_NETID “NetId”¶

<type>Int</type>

Boad number of the channel with the specified NetID. This setting is used, only if there is no setting for the board and channel number in the io configuration. Example: NetId.0.Board=0 // Channel with NetId = 0 is on Board 0 If board and channel number is not set, board 0 channel 0 is used for NetId 0.

#define NETXCMD_BOARD “Board”¶

#define NETXCMD_CHANNEL “Channel”¶

<type>Int</type>

Channel number of the channel with the specified NetID. This setting is used, only if there is no setting for the board and channel number in the io configuration. Example: NetId.0.Channel=0 // Channel with NetId = 0 is on NetX Channel 0
