Macro Definition Documentation¶

#define SYSFLASH_KEY_INT_ERASE_BLOCK_SIZE “EraseBlockSize”¶

Max. size of a block, which is erased in one piece. If a block exceeds this size, the erase operation is splitted up into several calls of SysFlashErase_() , each limited to this value. Should be set to a size, which is erased in below 1 second, because the CommCycleHook is called after each block to keep the rest of the RTS alive.

#define SYSFLASH_VALUE_INT_ERASE_BLOCK_SIZE_DEFAULT INT32_C(0x1000)¶
