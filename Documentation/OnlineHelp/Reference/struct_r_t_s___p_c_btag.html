5.3.633.2. Detailed Description¶

Pool Control Block. This is the header within a MemPool to manage it. ATTENTION:

  • This structure members are managed by the CmpMemPool and so don’t modify these entries directly!!!!!

5.3.633.3. Field Documentation¶

5.3.633.3.1. RTS_SIZE RTS_PCBtag::ulBlockSize¶

Block size of each pool block

5.3.633.3.2. PF_MEMBLOCK_DELETEFUNCTION RTS_PCBtag::pfDeleteFunction¶

Delete function pointer which is called optional to delete a MemPool block

5.3.633.3.3. RTS_I32 RTS_PCBtag::nRefCount¶

Reference counter which is incremented with MemPoolAddRef_LF() and is decremented with MemPoolReleaseRef_LF() to prevent, that the chain of used blocks is not changed during iteration over all elements.
