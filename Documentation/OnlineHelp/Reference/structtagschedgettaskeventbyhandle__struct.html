5.3.1515.2. Detailed Description¶

Function returns the handle to the task event. With this event a task can be activaed externally, e.g. for external triggered event tasks. The event can be sent by SysEventSet(EventHandle);

Parameters

[in] hSchedTask Scheduler task handle
[out] pResult ERR_OK or Error code

Returns:

Event handle. Can be used with SysEventSet(hEvent) of the SysEvent.library to wakeup the task
