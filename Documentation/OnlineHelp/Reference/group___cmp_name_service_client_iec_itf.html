5.2.68.7.4. typedef struct tagnsclientclose_struct nsclientclose_struct¶

Close an instance of the nameservice client. Callbacks that arrive after the client has been closed will no longer be forwarded. Returns the result of the operation, ERR_OK if a valid instance has been given in.

5.2.68.7.5. typedef struct tagnsclientopen_struct nsclientopen_struct¶

Opens an instance of the nameservice client. Returns the handle to the created name service client.

5.2.68.7.6. typedef struct tagnsclientresolveall_struct nsclientresolveall_struct¶

Initiates the resolving of all available runtimes. Returns the result of the operation.

5.2.68.7.7. typedef struct tagnsclientresponsecallback_struct nsclientresponsecallback_struct¶

This is only a prototype for the function that will be called when a node has been found. Only used to export the callback interface for generating the m4 runtime system files. There is no implementation behind that.... Must not be called!!!
