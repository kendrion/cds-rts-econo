#define RTS_TASKPREFIX_CAAASNYMANTASK “CAAAsnyManTask”¶

Task prefix to handle asynchronous task based jobs. The index is added to the prefix to get a unique task name. NOTE: CAA_ASM_ASYNC_TASK_PRIORITY is used for the task priority.
