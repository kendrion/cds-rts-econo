5.2.93.8. Macro Definition Documentation¶

5.2.93.8.1. #define CMPTARGETVISU_FLAGS_REMOTE 0x0001¶

Special flag that will be set when the targetvisualization is executed (ie. drawn) on a different plc than the paint commands are calculated. Probably only relevant when using CmpVisuHandlerRemote.

5.2.93.8.2. #define CMPTARGETVISU_FLAGS_REMOTE_BESTFIT 0x0002¶

Special flag that can be set when the targetvisualization is executed (ie. drawn) on a different plc than the paint commands are calculated. If this flag is set, then the visu will be drawn in best fit mode.

5.2.93.8.3. #define CMPTARGETVISU_FLAGS_MEASURE_PAINTCOMMANDEXECUTION 0x0004¶

Special flag that can be set when the targetvisualization should measure the time that is necessary for executing the paint commands. The result of this measurement is sent to the IEC visualization using an input event.

5.2.93.8.4. #define CMPTARGETVISU_FLAGS_MEASURE_MOUSEDOWNREACTION 0x0008¶

Special flag that can be set when the targetvisualization should measure the reaction time on mouse down events. the paint commands. The result of this measurement is sent to the IEC visualization using an input event. Mouse events that are raised faster than the reaction can be drawn cannot be measured using this mechanism

5.2.93.8.5. #define CMPTARGETVISU_FLAGS_MEASURE_PAINTINGTOSCREEN 0x0010¶

Special flag that can be set when the targetvisualization should measure the reaction time on mouse down events. the paint commands. The result of this measurement is sent to the IEC visualization using an input event. Mouse events that are raised faster than the reaction can be drawn cannot be measured using this mechanism

5.2.93.8.6. #define CMPTARGETVISU_FLAGS_ACTIVATE_TOUCHHANDLING 0x0020¶

Flag that activates the evaluation of touch events in the targetvisualization (of course only if supported by the according system implementation.

5.2.93.8.7. #define CMPTARGETVISU_FLAGS_ACTIVATE_KEYBOARD 0x0040¶

Flag that activates keyboard usage.

5.2.93.8.8. #define CMPTARGETVISU_FLAGS_REMOTE_SCALETYPEISOTROPIC 0x0080¶

Special flag that can be set when the targetvisualization is executed (ie. drawn) on a different plc than the paint commands are calculated. If this flag is set, then the visu will be drawn with scale type isotropic when best fit mode is set.

5.2.93.8.9. #define CMPTARGETVISU_FLAGS_REMOTE_BESTFIT_FOR_DIALOGS 0x0100¶

Special flag that can be set when the targetvisualization is executed (ie. drawn) on a different plc than the paint commands are calculated. If this flag is set, then the visu and the visualization dialogs will be drawn in best fit mode.

5.2.93.8.10. #define CMPTARGETVISU_NOTIFY_TAG_PAINTDATA_AVAILABLE 1¶

Notification that can be sent to the targetvisualization to signal that new paint data is available for execution. Arguments are not used at the moment.

5.2.93.9.2. typedef struct tagtargetvisucyclic_struct targetvisucyclic_struct¶

EXTERN LIB SECTION BEGIN

This function will be called cylically by the generic visualization libraries for active targetvisualization clients when it is supported by these libraries. The call is used for implementing a targetvisualization without the need for multitasking systems.

5.2.93.9.3. typedef struct tagtargetvisufindbyid_struct targetvisufindbyid_struct¶

5.2.93.9.4. typedef struct tagtargetvisunotify_struct targetvisunotify_struct¶

This function can be used to send notifications to a targetvisualization instance. As those notifications are usually sent in an asynchronous way, do not expect immediate effects when this function returned!
