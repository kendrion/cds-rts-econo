#define CMPSECCHANNEL_KEY_STRING_ONLINECOMMMODE CMPSETTINGS_CATEGORY_SECURITY “.CommunicationMode”¶

<type>String</type>

Communication mode to select, which kind of communication is possible. NOTE: Security modes must be ordered in an descending sorting, that the setting with the highest security level is found at the beginning of the list!
