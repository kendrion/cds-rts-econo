Detailed Description¶

EXTERN LIB SECTION BEGIN

Accept the next incoming TCP connection. | This functions calls the function accept of the operating system, which can accept a request to connect | to a socket. A new descriptor (handle) for the socket is returned. The original socket is reset to the | “listening” state (see SysSockListen).

For TLS Sockets the blocking mode will be set tononblockingfor the new client sockets. If a blocking behavior is needed use |SysSock2Ioctl| to set the socket to blocking mode. This reflects the behavior of |SysSock2Create| for TLS sockets.

return

Handle to the new accepted socket or RTS_INVALID_HANDLE if failed.
