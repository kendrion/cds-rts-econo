#define CMPRETAIN_NUM_OF_STATIC_RETAINS 8¶

Number of static retain memory entries to manage

#define CMPRETAIN_RETAIN_FILE_EXTENSION ”.ret”¶

File extension for storing retains

#define TAG_RETAIN_USED UINT32_C(0xABACADAE)¶

Tag types to detect the beginning of each retain area Retain area is in use

#define TAG_RETAIN_PERSISTENT_USED UINT32_C(0xABADAEAF)¶

#define TAG_RETAIN_FREE UINT32_C(0xEBECEDEE)¶

Retain area is free. Can be used for new areas

#define TAG_RETAIN_RESERVED UINT32_C(0xFBFCFDFE)¶

Retain area is reserved, because it was in use before the last shutdown. After complete reboot, this area is marked as free, if it is no longer in use.
