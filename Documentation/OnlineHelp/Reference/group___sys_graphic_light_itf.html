5.2.128.8.1. typedef struct tagsysgraphiclightbeginpaint_struct sysgraphiclightbeginpaint_struct¶

This function and SysGraphicEndPaint are used to encapsule some paint calls together. An implementing platform could use this encapsulation for some optimizations. For every call to SysGraphicBeginPaint there will be an according call to SysGraphicEndPaint.

Parameters

[in] hDisplayContext Handle to device context.
[in] rsCurrentVisuName The name of the current visualization.
[in] ulFlags Additional flags.

Returns:

*RESULT: Returns the runtime system error code (see CmpErrors.library).

5.2.128.8.2. typedef struct tagsysgraphiclightdrawbitmap_struct sysgraphiclightdrawbitmap_struct¶

Function to draw a bitmap into a rectangle.

Parameters

[in] hDisplayContext Handle to device context.
[in] rectangle Rectangle description. Bitmap data
[in] ulDataSize Bitmap data size
[in] ulTransparentColor Contains the color that should be used as the transparent color. Will only be evaluated when the according flag is set.
[in] ulFlags Flags. Defined in IEC-Code for Visualization. At the moment only the flag for transparency is relevant when loading an image!

Returns:

*RESULT: Returns the runtime system error code (see CmpErrors.library).

5.2.128.8.3. typedef struct tagsysgraphiclightdrawpolygon_struct sysgraphiclightdrawpolygon_struct¶

Function to draw a polygon based on an array of points.

Parameters

[in] hDisplayContext Handle to device context.
[in] iPoints Number of points in the array.
[in] pPoints Pointer to the array of points. Length must be less or equal nPoints!
[in] ePolygonType Polygon type. See corresponding category.

Returns:

*RESULT: Returns the runtime system error code (see CmpErrors.library).

5.2.128.8.4. typedef struct tagsysgraphiclightdrawrect_struct sysgraphiclightdrawrect_struct¶

Function to draw a rectangle based on top left and bottom right.

Parameters

[in] hDisplayContext Handle to device context.
[in] rectangle Rectangle description.
[in] eRectType Rectangle type. See corresponding category.

Returns:

*RESULT: Returns the runtime system error code (see CmpErrors.library).

5.2.128.8.5. typedef struct tagsysgraphiclightdrawtext_struct sysgraphiclightdrawtext_struct¶

Function to draw a text, based on top left and bottom right. Only the first line, if more than one is given.

Parameters

[in] hDisplayContext Handle to device context.
[in] rectangle Rectangle description.
[in] pszText

Will be either:

  • charif TF_WSTRING is not set in ulDrawFlags
  • shortif (TF_WSTRING is set in ulDrawFlags)
[in] ulDrawFlags Draw flags. Defined in IEC-Code for Visualization.

Returns:

*RESULT: Returns the runtime system error code (see CmpErrors.library).

This function and SysGraphicBeginPaint are used to encapsule some paint calls together. An implementing platform could use this encapsulation for some optimizations. For every call to SysGraphicBeginPaint there will be an according call to SysGraphicEndPaint.

Parameters

[in] rsCurrentVisuName The name of the current visualization.
[in] ulFlags Additional flags.

Returns:

*RESULT: Returns the runtime system error code (see CmpErrors.library).

5.2.128.8.7. typedef struct tagsysgraphiclightgetdisplaydevicecontext_struct sysgraphiclightgetdisplaydevicecontext_struct¶

Provides a handle to the basic display context. If you do not use this handle any more, release it using SysGraphicLightReleaseDisplayDeviceContext

Parameters

[in] ulDisplayID Optional parameter for indicating different display contexts by ID. If only one display is used the parameter is always 0.
[out] pResult Pointer to runtime system error code (see CmpErrors.library)

Returns:

*RESULT: Handle to the display context or RTS_INVALID_HANDLE if failed.
