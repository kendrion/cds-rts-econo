#define EVT_CmpMonitor2_PrepareWrite MAKE_EVENTID ( EVTCLASS_INFO , 1)¶

Event is sent to prepare the write operation to an expression. Note: this event is compatible with the event EVT_PrepareWriteVariable from the component CmpMonitor.
