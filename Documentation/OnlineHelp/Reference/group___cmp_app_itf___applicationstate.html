Macro Definition Documentation¶

#define AS_NONE UINT32_C(0x00000000)¶

  • AS_NONE: Unspecified state

  • AS_RUN: Application in run

  • AS_STOP: Application in stop

  • AS_DEBUG_HALT_ON_BP: Application halted on breakpoint

  • AS_DEBUG_STEP: Not used actually

  • AS_SINGLE_CYCLE: Application is executing a single cycle at the moment. After single cycle is operated, the application goes in status AS_STOP.

  • AS_SYSTEM_APPLICATION: State of a system application. State of a system application is masked and has no dedicated state.
