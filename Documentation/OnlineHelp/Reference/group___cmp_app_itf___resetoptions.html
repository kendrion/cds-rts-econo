Macro Definition Documentation¶

#define RTS_RESET 0¶

Warm reset. All global data except retain data is reset to their default values.

#define RTS_RESET_COLD 1¶

Cold reset. All global data AND retain data is reset to their default values.

#define RTS_RESET_ORIGIN 2¶

Origin reset. Delete the application, delete all application files (boot application, etc.), reset all global and retain data. After this command, the controller doesn’t know anything about the application.

#define RTS_RESET_ORIGIN_DEVICE 3¶

Origin reset device. Delete all applications, delete all application files (boot application, etc.), reset all global and retain data. Clear $visu$ and $PlcLogic$ folders. After this command, the controller doesn’t know anything about any application.
