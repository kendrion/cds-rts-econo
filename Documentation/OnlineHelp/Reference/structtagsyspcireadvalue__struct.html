Detailed Description¶

Read one PCI configuration value This function allows to read single configuration values of a PCI card by returning a pointer on the associated memory address (pbyData). Thereby the associated bus entry is identified via the bus, device and function number. The component to be read out is indicated by the PciOffset.

Returns:

RESULT: Returns the runtime system error code (see CmpErrors.library).
