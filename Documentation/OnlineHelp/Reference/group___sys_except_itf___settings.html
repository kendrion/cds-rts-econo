#define SYSEXCEPT_KEY_INT_ENABLE_FIRST_LEVEL_HANDLING “EnableFirstLevelHandling”¶

Setting to enable/disable first level exception handling: 1: enable first level handling 0: disable first level handling. Is done by an OEM component.
