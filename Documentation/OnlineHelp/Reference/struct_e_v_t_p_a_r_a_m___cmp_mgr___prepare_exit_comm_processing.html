Field Documentation¶

RTS_I32 EVTPARAM_CmpMgr_PrepareExitCommProcessing::bLastCall¶

This value will be != 0 for the last call of this event, otherwise 0.

RTS_I32 EVTPARAM_CmpMgr_PrepareExitCommProcessing::bFurtherCallNecessary¶

Callees of the event can set this value to != 0 if they expect a further call of the same event with in between comm cycle calls. It is not expected that a callee sets this value to 0!
