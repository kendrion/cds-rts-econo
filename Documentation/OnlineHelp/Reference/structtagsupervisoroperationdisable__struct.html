7.1.5.1937. tagsupervisoroperationdisable_struct Struct Reference¶

#include “CmpSupervisorItf.h”

Data Fields¶

  • RTS_IEC_HANDLE hOperation

  • RTS_IEC_RESULT SupervisorOperationDisable

Detailed Description¶

Disables supervision for this operation. This operation will never be supervised, until the next SupervisorOperationEnable() call!

return

Error code

Error code: + ERR_OK: Supervision was successfully disabled + ERR_NOTINITIALIZED: The operation memory is not initialized + ERR_INVALID_HANDLE: The handle to the operation is invalid + ERR_PARAMETER: The handle to the operation is invalid + ERR_NO_CHANGE: The operation is already disabled
