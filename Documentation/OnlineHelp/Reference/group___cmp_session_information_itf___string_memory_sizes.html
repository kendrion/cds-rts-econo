Macro Definition Documentation¶

#define SESS_INFO_MEMSIZE_USER_NAME UINT16_C(64)¶

Number of bytes of stings including the 0 terminator. Memory size of the string for the user logged in on the session.

#define SESS_INFO_MEMSIZE_CLIENT_NAME UINT16_C(256)¶

Memory size of the string for the name of the client application that created that session.

#define SESS_INFO_MEMSIZE_CLIENT_VENDOR_NAME UINT16_C(256)¶

Memory size of the string for the name of the client application’s vendor.

#define SESS_INFO_MEMSIZE_CLIENT_HOST_NAME UINT16_C(256)¶

Memory size of the string for the host name where client is running.

#define SESS_INFO_MEMSIZE_CLIENT_VERSION UINT16_C(256)¶

Memory size of the string for the client version.

#define SESS_INFO_MEMSIZE_CLIENT_CDS_VERSION UINT16_C(256)¶

Memory size of the string for the CODESYS version the client is based on.
