Detailed Description¶

ONLY FOR INTERNAL TEST!

Processor load task to detect, if IEC tasks consumes too much CPU.

Exception task to handle all exceptions in the IEC tasks.

Scheduletask to handle external trimeslicing.

Scheduletask to handle cyclic tick at external trimeslicing.

Scheduletask to handle schedule tick.

Scheduletask to handle the task gap.
