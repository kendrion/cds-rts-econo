5.3.44.1. Data Fields¶

  • OpcUa_NodeId startNode
  • OpcUa_NodeId destinationNode
  • OpcUa_NodeId referenceType
  • OpcUa_Boolean bBidirectional

5.3.44.2. Detailed Description¶

This structure describes a reference that should be added to the OPC UA server. All IDs in this structure have to known to the OPC UA server.

5.3.44.3. Field Documentation¶

5.3.44.3.1. OpcUa_NodeId _OpcUaServerReferenceDescription::startNode¶

ID of the node where the reference starts.

5.3.44.3.2. OpcUa_NodeId _OpcUaServerReferenceDescription::destinationNode¶

ID of the node where the reference points to.

5.3.44.3.3. OpcUa_NodeId _OpcUaServerReferenceDescription::referenceType¶

ID of the RefereceType to be used.

5.3.44.3.4. OpcUa_Boolean _OpcUaServerReferenceDescription::bBidirectional¶

Flag if the reference should be added in both directions.
