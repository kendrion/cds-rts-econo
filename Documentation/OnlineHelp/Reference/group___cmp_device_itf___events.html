#define EVT_CmpDevice_InteractiveLogin MAKE_EVENTID ( EVTCLASS_INFO , 1)¶

Event is sent to operate the interactive login or the wink feature. 1. Wink: The user can select a device in the Gateway dialog, and let it “blink”. This allows to identify a device in the list of devices found in the scan. This may be expecially neccessary while PLC don’t have unique or meaningful names yet. 2. Secure Login: During login, it can be guaranteed that the user is connecting to the correct PLC.

Parameters

[in] pEventParam EVTPARAM_CmpDevice_InteractiveLogin
