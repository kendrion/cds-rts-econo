#define IEC_TASK_STOPREASON_OPTION_PREPARE_SAFE_STATE 0x00000000¶

Option that can be specified for IecTasksWaitForStop() in stop reason Stop all IEC tasks _before_ setting the outputs to a safe state

#define IEC_TASK_STOPREASON_OPTION_DONE_SAFE_STATE 0x00010000¶

Stop all IEC tasks _after_ setting the outputs to a safe state. Here we can wait a longer period!
