#define CMPASYNCMGR_INT_DEFAULT_TASK_PRIORITY_IEC “DefaultTaskPriorityIEC”¶

Setting to specify the default task priority of the asynchronous task created from IEC. NOTE: This priority is used if the caller has not specified any priority (or if it is set to 0).
