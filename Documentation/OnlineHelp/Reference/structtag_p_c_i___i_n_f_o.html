5.3.1434.2. Detailed Description¶

PCI information entry of one device.

Parameters

[in] usVendorID Registered Vendor ID of card assigned by PCI SIG
[in] usDeviceID Device ID of card assigned by manufacturer
[in] usSubVendorID Registered subsystem vendor ID assigned by PCI SIG
[in] usSubSystemID Subsystem device ID assigned by manufacturer, if there are multiple subsystems on card. The value 0 indicates that there are no subsystems.
[in] ulBusNr PCI bus number the PCI card is member of
[in] SlotNr PCI slot number (within bus) the PCI card is plugged in
[in] ulFunction PCI function number
[in] ulBaseAddresses Basis address register for reservation of I/O port addresses or I/O memory areas (Memory Mapped I/O); Specific for particular card.
[in] byInterrupt Identifies the interrupt vector on the particular system
[in] DeviceSpecific According to PCI specification the 192 manufacture specific data bytes
