Interface of CmpAsyncMgr.

Detailed Description¶

Interface of the asynchronous job manager. This job manager enables, to execute synchronous jobs asynchronously.

RTS_HANDLE AsyncAdd (PFASYNCJOBFUNCTION pfAsyncJobFunc, void *pParam, void *pInstance, RTS_UI32 *pulState, RTS_I32 bIecFunc, RTS_UI32 ulType, RTS_UI32 ulTimeout, ASYNCJOB_PARAM *pAsyncJobParam, RTS_RESULT *pResult)¶

EXTERN LIB SECTION END

This function adds a new asynchronous job in a job queue

Parameters

[in]

pfAsyncJobFunc

Pointer to the job function, which is executed asynchronously

[in]

pParam

Pointer to the parameters for the function

[in]

pInstance

Optional instance point, if the callback is an IEC function block

[in]

pulState

Pointer to actual state of the asynchronous job. NOTE: Can be NULL for fire and forget jobs, which are released automatically at the end of job execution!

[in]

bIECFunc

TRUE=pfAsyncJobFunc is an IEC function, FALSE=pfAsyncJobFunc is a C-function

[in]

ulType

Type of asynchronous job (task, event or hook driven)

[in]

ulTimeout

Timeout in [milliseconds], until the job must be started! If the job is not started within this timeout, the state of the job is marked with ASYNCSTATE_TIMEOUT and the job is removed from the queue.

[in]

pAsyncJobParam

Pointer to specific parameter for the context of the job execution (task, event or hook driven)

[out]

pResult

Error code

Returns:

Handle to the job object
