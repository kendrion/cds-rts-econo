Interface of CmpUserObjectsDB.

Detailed Description¶

Interface for the user rights management database.

Here you have to store the tree of all objects of the runtime system with their assigned user-groups and the configured rights on it.

Example:

  • Object: Name = “Device”; UserGroup = “Administrator” | Rights = USERDB_RIGHT_ALL | DeniedRights = USERDB_RIGHT_NONE; UserGroup = “…” | Rights = … | DeniedRights = …; …

  • Object: Name = “Device.MyObject”; UserGroup = “Administrator” | Rights = USERDB_RIGHT_ALL | DeniedRights = USERDB_RIGHT_NONE; UserGroup = “…” | Rights = … | DeniedRights = …; …

For details see CmpUserMgrItf.h .
