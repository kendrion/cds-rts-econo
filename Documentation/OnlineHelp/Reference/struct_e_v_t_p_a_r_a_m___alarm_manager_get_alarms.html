5.3.156.3.1. RTS_IEC_STRING* EVTPARAM_AlarmManagerGetAlarms::pszApplName¶

The name of the application, which has to process the request.

5.3.156.3.2. RTS_IEC_INT EVTPARAM_AlarmManagerGetAlarms::iSkipRows¶

An int value with the number of rows, that have to skipped.

5.3.156.3.3. RTS_IEC_INT EVTPARAM_AlarmManagerGetAlarms::iMaxRows¶

An int value with the maximum number of rows, that have to be returned.

5.3.156.3.4. RTS_IEC_UDINT EVTPARAM_AlarmManagerGetAlarms::udiCountAlarmGroups¶

The number of alarm group IDs the parameter puiAlarmGroupIDs points to. If no alarm group IDs are specified, this parameter is 0.

5.3.156.3.5. RTS_IEC_UDINT* EVTPARAM_AlarmManagerGetAlarms::pudiAlarmGroupIDs¶

Points to an array of UDINT values with the IDs of the alarm groups to use as filter. If no such filter has been specified this parameter is NULL.

5.3.156.3.6. RTS_IEC_UDINT EVTPARAM_AlarmManagerGetAlarms::udiCountAlarmClasses¶

The number of alarm class IDs the parameter puiAlarmClassIDs points to. If no alarm class IDs are specified, this parameter is 0.

5.3.156.3.7. RTS_IEC_UDINT* EVTPARAM_AlarmManagerGetAlarms::pudiAlarmClassIDs¶

Points to an array of UDINT values with the IDs of the alarm classes to use as filter. If no such filter has been specified this parameter is NULL.

5.3.156.3.8. RTS_IEC_USINT EVTPARAM_AlarmManagerGetAlarms::usiPriorityFrom¶

A byte value with minimum priority the alarm must have.

5.3.156.3.9. RTS_IEC_USINT EVTPARAM_AlarmManagerGetAlarms::usiPriorityTo¶

A byte value with maximum priority the alarm must have.
