RTS_IEC_UDINT EVTPARAM_AlarmManagerGetAlarms::udiCountAlarmGroups¶

The number of alarm group IDs the parameter puiAlarmGroupIDs points to. If no alarm group IDs are specified, this parameter is 0.

RTS_IEC_UDINT* EVTPARAM_AlarmManagerGetAlarms::pudiAlarmGroupIDs¶

Points to an array of UDINT values with the IDs of the alarm groups to use as filter. If no such filter has been specified this parameter is NULL.

RTS_IEC_UDINT EVTPARAM_AlarmManagerGetAlarms::udiCountAlarmClasses¶

The number of alarm class IDs the parameter puiAlarmClassIDs points to. If no alarm class IDs are specified, this parameter is 0.

RTS_IEC_UDINT* EVTPARAM_AlarmManagerGetAlarms::pudiAlarmClassIDs¶

Points to an array of UDINT values with the IDs of the alarm classes to use as filter. If no such filter has been specified this parameter is NULL.
