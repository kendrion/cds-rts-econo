#define SYSMODULEKEY_STRING_VXWORKS_LOAD_MODULE_HEAP “VxWorks.LoadModuleHeap”¶

<type>String</type>

Setting to set VxWorks loadModule() options: Proximity: loadModule( ) with default / proximity heap flag Common: loadModule( ) with common heap flag (only available for VxWorks 7)
