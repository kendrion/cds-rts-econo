Interface of CmpMonitor.

Detailed Description¶

Interface of the monitoring component, that provides monitoring of IEC variables.

NOTE:

  • Use CmpMonitor2 instead

  • If you have to support previous CODESYS version before v3.5.0.0, this component must be integrated in parallel to CmpMonitor2!

RTS_RESULT MonitoringHasFeature (unsigned long ulFeatures)¶

Parameters

[in]

ulFeatures

Feature flags, See corresponding category “Features”.

Returns:

ERR_OK if the flags are supported, an error code otherwise

RTS_RESULT MonitoringReadValue (APPLICATION *pappl, BINTAGREADER *preader, RTS_UI8 *pbyValue, RTS_UI16 usSize)¶

Routine to read a value via a monitoring service. Note: this function is equivalent to calling MonitoringReadValue2 with bIec = 0.

Parameters

[in]

pappl

Pointer to application.

[in]

preader

the reader specifying the location of the value to monitor.

[in]

pbyValue

pointer to the destination address were the retrieved value is written.

[in]

usSize

size of value to read.

Returns:

ERR_OK if the value could be retrieved successfully
