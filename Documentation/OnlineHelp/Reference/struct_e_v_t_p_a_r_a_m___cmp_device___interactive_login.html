5.3.207.3.1. RTS_UI32 EVTPARAM_CmpDevice_InteractiveLogin::ui32LoginCommand¶

Login command. See category “Interactive login commands” (look for DEV_ILC_xxx defines).

5.3.207.3.2. RTS_UI32 EVTPARAM_CmpDevice_InteractiveLogin::ui32LoginMode¶

Login mode. See category “Interactive login mode flags” (look for DEV_ILMF_xxx defines). Only used for command DEV_ILC_LOGIN_REQUEST.

5.3.207.3.3. RTS_UI8* EVTPARAM_CmpDevice_InteractiveLogin::pui8LoginData¶

Login information to be checked by the runtime system. Only used for mode DEV_ILM_STRING

5.3.207.3.4. RTS_RESULT EVTPARAM_CmpDevice_InteractiveLogin::rLoginResult¶

Error code of event handler:

  • ERR_OK: Operation succeeded
  • ERR_ID_MISMATCH: Is returned at DEV_ILMF_ID, if the ID does not match (e.g. the entered serial number is not identical to the target)
  • ERR_PENDING: Is returned at DEV_ILMF_KEY_PRESS, if we wait for pressing the key
