#define EDGESV_HANDSHAKE_HELLO 0x01¶

Service group: EDGESG_HANDSHAKE

Service: EDGESV_HANDSHAKE_HELLO

Request tags:

  • Tag Max protocol version (1..?) (RTS_UI32): Maximum protocol version supported by the edge gateway (high word: major version, low word: minor version).

  • Tag Target type (1..?) (RTS_UI32): Type of the device which contains the edge gateway.

  • Tag Target ID (1..?) (RTS_UI32): ID of the device containing the edge gateway.

  • Tag Target version (1..?) (RTS_UI32): Version of the device containing the edge gateway.

  • Tag Core version (1..?) (RTS_UI32): Version of the CmpEdgeGateway.

  • Tag Max message size (1..?) (RTS_UI32): Maximum supported frame size of the edge gateway.

  • Tag Connection timeout (1..?) (RTS_UI32): Default channel timeout of the edge gateway.

  • Tag Certificate (1..?) (String): Base64 encoded edge certificate.

Response tags:

  • Tag Effective protocol version (1..?) (RTS_UI32): Effective protocol version used on this channel (high word: major version, low word: minor version).

  • Tag Effective message size (1..?) (RTS_UI32): Effective frame size used on this channel.

  • Tag Effective connection timeout (1..?) (RTS_UI32): Effective timeout of this channel.

  • Tag Signature algorithm (1..?) (RTS_UI32): Identifies the algorithm used by the edge to sign the nonce.

  • Tag Server nonce (1..?) (ByteArray): A random nonce generated by the automation server.

  • Tag Group reference (1..?) (RTS_UI64): Group reference.

  • Tag Number of apps (0..?) (RTS_UI32): Number of applications.

  • Tag App name (0..?) (MemberString): Application name.

  • Tag App deleted (0..?) (Empty): Application has been removed.

  • Tag Last change (0..?) (RTS_UI32): ulLastChanges (UTC time in seconds since 1.1.1970 00:00:00).

  • Tag Project (0..?) (MemberString): pszProject.

  • Tag Version (0..?) (MemberString): pszVersion.

  • Tag Author (0..?) (MemberString): pszAuthor.

  • Tag Description (0..?) (MemberString): pszDescription.

  • Tag Profile (0..?) (MemberString): pszProfile.

  • Tag Code GUID (0..?) (ByteArray): CodeGuid.

  • Tag Data GUID (0..?) (ByteArray): DataGuid.

  • Tag App status (0..?) (RTS_I16): New application status as value of enumeration type PLC_STATUS_ENUM.

#define EDGESV_VAR_DEFINE_VARLIST 0x01¶

Service group: EDGESG_PLC_VAR_ACCESS

Service: EDGESV_VAR_DEFINE_VARLIST

Request tags:

  • Tag PLC reference (1..?) (RTS_UI64): PLC reference (0=Invalid).

  • Tag Variable list reference (1..?) (RTS_UI64): Variable list reference.

  • Tag Update rate (0..?) (RTS_UI32): Update rate for the cyclic update of the variable values in ms (default: 200).

  • Tag Flags (0..?) (RTS_UI32): Flags for the variable list (e. g. consistency flags).

  • Tag Number of symbols (1..?) (RTS_UI32): Number of symbols in the list.

  • Tag Symbols (1..?): List of symbols as array (the number of array elements has to match to the number of symbols). Each array element has following members: [RTS_UI32]: ulTypeId (see PlcSymbolDescStruct.ulTypeId) [RTS_UI32]: ulSize (see PlcSymbolDescStruct.ulSize) [MemberString]: pszSymbol (name of the symbol)

Notification tags:

  • Tag Var list reference (1..?) (RTS_UI64): Variable list reference.

  • Tag Index (0..?) (RTS_UI32): Start index of the symbol in the variable list (default: 0).

  • Tag Read values (1..?): List of Values (members of PlcVarValueStruct as stream with content as follows): [RTS_UI32]: ulTimeStamp (Seconds since 1.1.1970 00:00:00 UTC) [RTS_UI8]: bQuality [ByteArray]: byData
