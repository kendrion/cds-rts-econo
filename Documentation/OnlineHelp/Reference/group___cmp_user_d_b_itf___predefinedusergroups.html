#define USERDB_GROUP_ADMINISTRATOR “Administrator”¶

Predefined user groups Administrator group. A member of this groups has all access rights on all objects!

#define USERDB_GROUP_DEVELOPER “Developer”¶

Developer group. A member of this groups has all access to the IEC applications an all related stuff.

#define USERDB_GROUP_SERVICE “Service”¶

Service group. A member of this groups has all access to object for maintenance and service operations.

#define USERDB_GROUP_WATCH “Watch”¶

Watch group. A member of this groups has read-only access to the objects.

#define USERDB_GROUP_EVERYONE “Everyone”¶

Everyone group. Every user is automatically member of this group! NOTE: By default, this group has read-only access to the objects. But this can be deactivated, so every user must be a member of one of the groups above and have no access right by default!
