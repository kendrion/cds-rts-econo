Detailed Description¶

: This structure can be used to get and set the settings of a COM port.

NOTE: The difference of this structure to SysComSettingsEx are the 2 members byDtrControl and byRtsControl! Here you can specify BYTE instead of BOOL values in SysComSettingsEx!
