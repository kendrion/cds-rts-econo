Macro Definition Documentation¶

#define PVF_FUNCTION 0x0001¶

Defines the type of the parameter

Actually only PVF_POINTER is currently supported by CoDeSys Functional access to the parameter, Value is a function pointer.

#define PVF_POINTER 0x0002¶

Value is a pointer to the value

#define PVF_VALUE 0x0004¶

Obsolete: NOT USED. Value can be interpreted directly as a value (max. 32 Bit data types)

#define PVF_READ 0x0010¶

Parameter can be read

#define PVF_WRITE 0x0020¶

Parameter can be written
