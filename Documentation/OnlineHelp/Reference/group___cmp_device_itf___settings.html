Macro Definition Documentation¶

#define DEVKEY_INT_OPERATION_MODE “OperationMode”¶

<type>Int</type>

Setting to store the current operation mode of the PLC. For possible values see category Device operation mode. Not a configuration option - only to be set/changed by the runtime system.

#define DEVVALUE_INT_OPERATION_MODE_DEFAULT DEV_OM_DEBUG¶

#define DEVKEY_INT_OPERATION_MODE_CHECK “OperationModeCheck”¶

<type>Int</type>

Setting to store some additional information for some operation modes, to be checked at startup. Not a configuration option - only to be set/changed/removed by the runtime system.
