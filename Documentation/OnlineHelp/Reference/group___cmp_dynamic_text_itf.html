5.2.38.7. Typedef Documentation¶

5.2.38.7.1. typedef struct tagdynamictextloaddefaulttexts_struct dynamictextloaddefaulttexts_struct¶

This function loads the default texts from the currently registered text files. as a IEC-String (charin the runtime)

5.2.38.7.2. typedef struct tagdynamictextreloadtexts_struct dynamictextreloadtexts_struct¶

This function reloads all currently registered text files for the currently assigned language. This function could be used for example when you know that a textfile was updated externally and you want to reload those files.

5.2.38.7.3. typedef struct tagdynamictextgetcurrentlanguage_struct dynamictextgetcurrentlanguage_struct¶

This method returns the name of the currently loaded language if there is any.

If no language is loaded (there was no call to DynamicTextChangeLanguage before), then an empty string will be returned.

5.2.38.7.4. typedef struct tagdynamictextregisterfile_struct dynamictextregisterfile_struct¶

This function registers a dynamic text file. These registered text files are considered when load or reload is called on the CmpDynamicText component.

5.2.38.7.5. typedef struct tagdynamictextunregisterfile_struct dynamictextunregisterfile_struct¶

This function unregisters a dynamic text file. The registered text files are considered when load or reload is called on the CmpDynamicText component.

5.2.38.7.6. typedef struct tagdynamictextregisterpath_struct dynamictextregisterpath_struct¶

This function sets an optional path where the dynamic text files are searched when they are loaded or reloaded by the CmpDynamicText component.
