RTS_UINTPTR AppDebugHandler (RTS_UINTPTR IP, RTS_UINTPTR SP, RTS_UINTPTR BP)¶

Routine handle a debug breakpoint. This routine is typically called from the syscpudebughandler() method of the SysCpuHandling component. This function handles all debugging features (entering/leaving a breakpoint, set next statement, flowcontrol, executionpoints, etc.)

Parameters

[in]

IP

Program counter or instruction pointer of breakpoint position, where the debug opcode is executed

[in]

SP

Actual stack pointer

[in]

BP

Frame pointer or base pointer to the stack, where the first entry function right after the debug code begins its stack usage

Returns:

Return address, to which we would like to return right after leaving this function. NOTE: This must be adopted by the caller!
