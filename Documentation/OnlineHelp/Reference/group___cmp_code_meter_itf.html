5.2.32.4.3. #define RTS_CODEMETER_LICENSES_FIRMCODE 101597¶

5.2.32.4.4. #define RTS_CODEMETER_ENCRYPTION_FIRMCODE 101599¶

Firmcode for encryption of the bootproject

5.2.32.4.5. #define CODEM_3S_INIT_LICENSE_FILE “3SLicense.wbb”¶

This is the name of the 3S-wbb file to create an empty softcontainer!

5.2.32.4.6. #define CODEMKEY_STRING_ACTLICENSE_DIRECTORY “ActLicenseDirectory”¶

Setting to specify the directory for all act license files.

5.2.32.4.7. #define CODEMVALUE_STRING_ACTLICENSE_DIRECTORY_DEFAULT ”./cmact_licenses”¶

5.2.32.4.8. #define CODEMKEY_INT_INSTALL_CMACT_CONTAINER “InstallCmActContainer”¶

Setting to activate installing license container. The wbb files(s) must be registered for that (see setting “InitLicenseFile”). NOTE: The key is deleted after installing the container!
