Macro Definition Documentation¶

#define TRACE_PACKET_FLAGS_NOT_INITIALIZED 0x00000000¶

Trace packet is not initialized

#define TRACE_PACKET_FLAGS_COMPLETED 0x00000001¶

Sign, that the trace packet is completed (all records added)

#define TRACE_PACKET_FLAGS_AUTOSTART 0x00000002¶

Persistent storage of the trace, an automatic start of the trace after reboot or to restore only on request

#define TRACE_PACKET_FLAGS_CONDITION 0x00000004¶

Optional boolean condition for the trace packet specified

#define TRACE_PACKET_FLAGS_TIMESTAMP_MS 0x00000010¶

Timestamp for the trace values have millisecond resolution (ticks)

#define TRACE_PACKET_FLAGS_TIMESTAMP_US 0x00000020¶

Timestamp for the trace values have microsecond resolution (ticks)

#define TRACE_PACKET_FLAGS_STORE_IN_FILE 0x00000040¶

Is used to initiate to store trace persistent in file. Only used, if TRACE_PACKET_FLAGS_AUTOSTART is set.

#define TRACE_PACKET_FLAGS_SYSTEM_TRACE 0x00010000¶

Is used to mark a trace as system trace.
