#define MAX_LEN_APPLNAME 60¶

Maximum length fo application name

#define APPL_NUM_OF_STATIC_APPLS 8¶

Length of application list that is allocated static

#define APPL_NUM_OF_STATIC_SESSIONIDS 5¶

Length of session id list that is allocated static

#define APPL_NUM_OF_STATIC_AREAS 10¶

Length of area pointer list that is allocated static in the application object

#define APP_NUM_OF_STATIC_ASYNC_SERVICES 8¶

Number of possible static async services. Can be increased dynamically.

#define APP_UDINT_COMPILERVERSION_DEFAULT 0x03000000¶

The default compiler version. Programming systems < 3.5.12.0 do not provide the compiler version on download. In this case the earliest version to expect is 3.0.0.0.

#define APP_BOOTPROJECT_FILE_EXTENSION ”.app”¶

Bootproject file extension

#define APP_BOOTPROJECT_FILE_EXTENSION_INVALID ”.ap_”¶

Bootproject file extension to invalidate during download

#define APP_BOOTPROJECT_FILE_EXTENSION_ERROR ”.err”¶

Bootproject file extension if file has an error

#define APP_BOOTPROJECT_FILE_EXTENSION_CRC ”.crc”¶

File extension for bootproject crc checksum file

#define APP_FILE_EXTENSION_SYMBOLS ”.xml”¶

File extension for symbol file matching to application

#define APP_BOOTPROJECT_FILE_EXTENSION_SYMBOLS ”.boot.xml”¶

File extension for symbol file matching to bootproject

#define CCO_SEMAPHORE 1¶

Defines specifies the synchronization mechanism of the online change code to the IEC tasks. CCO_DEFAULT defines the option, which mechanism is used to synchronize the online change code (copy code). OnlineChange code is synchronized via a semaphore

#define CCO_TASK_GAP 2¶

OnlineChange code is executed in the IEC task gap in the context of the scheduler tick (recommended machnism)

#define CCO_INTERRUPT_LOCK 3¶

OnlineChange code is synchronized via interrupt locks (most invasive mechanism)

#define CCO_TASK_GAP_SEMAPHORE 4¶

OnlineChange code is synchronized via a semaphore and is executed in the IEC task in the context of the scheduler tick

#define CCO_DEFAULT CCO_TASK_GAP_SEMAPHORE¶
