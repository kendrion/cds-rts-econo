Detailed Description¶

Prototypical implementation of the depictor for the targetvisualization. This component relies on the SysNativeControl usable within the Targetvisualization and uses the Irrlicht-Engine as basis for a 3D-Visualization-Element within CODESYS. Right now this is a proto type (as the corresponding plugin for the programming system is). Consequently no other components rely on this one or use it.
