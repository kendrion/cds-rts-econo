Detailed Description¶

EXTERN LIB SECTION END

Task info Task information out of the task configuration Note for SIL2: This information is already inside of the data area, so we don’t need to tag it as safe anywhere.

Note: This struct is used in CODESYS (AKA programming system) as well as in CODESYSControl (AKA run-time system). Whenever this struct is changed, a new version for its member dwVersion has to be used.
