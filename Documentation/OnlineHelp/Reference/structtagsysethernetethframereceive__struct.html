7.1.5.1991. tagsysethernetethframereceive_struct Struct Reference¶

  • RTS_IEC_HANDLE hAdapter

  • SysEthernetFrame * SysEthernetEthFrameReceive

Detailed Description¶

This function receives the next Ethernet frame from the Ethernet adapter. A pointer to this frame is returned. ..note:: The frame returned has to be released using the SysEthernetEthFrameRelease() function. The frame will only be overwritten if the frame is released. As frames are reused when released pay attention to call this function if processing of the frame has finished. :return: Pointer to the Ethernet frame
