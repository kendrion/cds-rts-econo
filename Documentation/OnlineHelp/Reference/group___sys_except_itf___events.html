#define EVT_EXCPT_GenerateException MAKE_EVENTID ( EVTCLASS_EXCEPTION , 1)¶

The event is sent if an exception occurred (see category exception code for detailed information). NOTE: This event is fired for every exception! That means also for handled exceptions via SEH (Structured Exception Handling) with rts_try/rts_catch!

Parameters

[in]

pEventParam

EVTPARAM_SysExcept

#define EVT_EXCPT_GenerateException2 MAKE_EVENTID ( EVTCLASS_EXCEPTION , 2)¶

The event is sent if an exception occurred (see category exception code for detailed information). NOTE: This event is fired only for exceptions that are not handled via SEH (Structured Exception Handling) with rts_try/rts_catch!

Parameters

[in]

pEventParam

EVTPARAM_SysExcept
