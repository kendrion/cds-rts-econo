#define MEM_POOL_OPTION_STATIC 0x00000001¶

MemPool options MemPool based on a static array

#define MEM_POOL_OPTION_DYNAMIC 0x00000002¶

MemPool based on a dynamic allocated heap block

#define MEM_POOL_OPTION_NOSYNC 0x00000010¶

No synchronization necessary within the MemPool

#define MEM_POOL_OPTION_UNALIGNED 0x00000100¶

Marks the MemPool as unaligned, if the beginning of the aligned pPool is not identical to the pHeapMemory pointer!
