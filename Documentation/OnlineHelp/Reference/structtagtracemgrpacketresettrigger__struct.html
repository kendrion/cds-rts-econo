7.1.5.2354. tagtracemgrpacketresettrigger_struct Struct Reference¶

#include “CmpTraceMgrItf.h”

Data Fields¶

  • RTS_IEC_HANDLE hPacket

  • RTS_IEC_RESULT TraceMgrPacketResetTrigger

Detailed Description¶

Resets the trigger of a trace packet. The start time and date of of the trigger is reset to 0 and the trigger is set to state enabled (if it is not disabled).

return

Returns the runtime system error code (see CmpErrors.library):

  • ERR_PARAMETER if hPacket is not a valid packet handle
