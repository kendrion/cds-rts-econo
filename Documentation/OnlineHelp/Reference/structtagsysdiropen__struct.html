Detailed Description¶

Opens a specified directory and returns a handle and the first directory entry. | szDirEntry is expected to provide enough space to write the directory entry to. | In case szDirEntry is 0, the first directory entry may be retrieved with SysDirRead. :return: Returns the runtime system error code (see CmpErrors.library)
